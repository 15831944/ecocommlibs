MyScript Analyzer

  • Technology
  • Analyzer

    • MyScript Analyzer is designed for building applications using a digital ink input with mixed content of handwritten text and drawn objects: It classifies the elements that make up a document as categories (text, shapes, and so forth). Then, it drives them to the appropriate recognizer type for conversion into digitized document. It can be used typically in note-taking applications.

    Features

    Elements that can be identified

    MyScript Analyzer can identify text and geometric shapes, and within text, text lines, paragraphs, bulleted lists, tables, underlined text, corrected text (strikethrough, cross-out, scratch-out).

    Layout

    MyScript Analyzer also identifies the layout of the page: text located inside a geometric shape, text located inside a table cell, alignment of different text blocks.

    Beautification

    MyScript Analyzer provides beautification information for the page by adjusting the positions and size of the different elements, in order to make the interpreted results appear more regular and normalized.

    Supported items

    MyScript Analyzer is able to distinguish the following items:

    Text line

    This is a single line of text.

    Table

    This is defined as a group of cells with some text inside.

    Shape

    This is defined as a single geometric shape recognized by the MyScript Shape technology.

    Group

    This is a group of items brought together by a common characteristic:

    • Paragraph: a coherent set of text-lines.
    • List: a bulleted list of text-lines.
    • Text inside graphic: a text-line or a paragraph that is inside a shape or a table’s cell.

    Correction

    Three types of corrections can be detected: scratch-outs (a scribbled line, made over a piece of text), strikethrough (a single line, made over a piece of text) and cross-outs (a pair of crossed lines, made over a piece of text).
