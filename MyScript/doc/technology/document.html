MyScript Document Model

  • Technology
  • Document Model

    • MyScript Document Model allows recognition processes on the ink data contained in a document.

    Features

    Document file format

    MyScript Document Model introduces a new document file format. A document can contain a collection of objects (text or binary files, e.g. images, audio files, scripts, html files, etc.), a sequence of pages and metadata information. MyScript Document Model allows recognition processes on the ink data contained in a document.

    Creating and editing documents

    MyScript Document Model can be used to create and edit complex documents. It is possible to create different content field types (text, shape, math, music, multimedia objects).

    Editing digital ink

    Digital ink can be edited. This refers to the ability to perform operations such as transformations (rotation, translation and so forth), deletion, undo/redo, and so on.

    Layout and rendering control

    MyScript Document Model technology allows manipulating the layout and the rendering of pages.

    Typesetting

    MyScript Document Model provides tools for typesetting recognized text or shape content.

    MyScript technologies

    MyScript Document Model is meant to be used in conjunction with MyScript Text, MyScript Shape, MyScript Math, MyScript Music, MyScript InkSearch and/or MyScript Analyzer. Potentially, it can be used in isolation, alone, for notes and document edition, storage and exchange.
