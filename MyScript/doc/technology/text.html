MyScript Text

  • Technology
  • Text

    • MyScript Text is designed for building applications that enable the recognition of handwritten text. It is ideal for embedding natural handwriting into note or form-based applications.

    Features

    Languages

    MyScript Text supports 65 languages in cursive script and 97 in isolated character mode, across 13 distinct character sets.

    Alphabets

    MyScript Text supports the following alphabets: Arabic, Farsi and Urdu characters, Armenian characters, CJK (Chinese, Japanese, Korean), East Asian characters, Devanagari characters, Georgian characters, Greek characters, Hebrew characters, Latin characters, Tamil characters, Thai characters.

    Handwriting style

    MyScript Text supports the recognition of isolated character, cursive, print and superimposed writing.

    Correction in real time

    MyScript Text allows users to edit in real time using natural gestures, including scratch-outs / strikethroughs, text insertions, and more. With the convenience of handwriting and the power of word processing functionality, entering and editing data has never been quicker or easier.

    Connection with related knowledge

    MyScript Text is able to identify items within the handwritten text, like dates, phone numbers, or names. It allows building advanced user interactions based on handwritten text, such as adding content to the calendar or to contacts, calling a phone number, searching the Web for a name, etc.

    Extending resources recognition power

    MyScript Text allows fine-tuning the recognition process power by creating custom resources. Easy-to-use tools are provided for creating, compiling and managing custom resource files.

    Gesture recognition

    MyScript Text supports gesture recognition: left, right, down then left, down then right gestures.

    Advanced capabilities

    • InkSearch: MyScript InkSearch enables users to search into their handwritten notes and receive immediate, precise results. The search function can identify the searched text within pages or documents that contain indexed handwritten content.

    • Writer Adaptation: MyScript Writer Adaptation learns individual users’ unique writing style and habits, making interpretation simple and reliable even in cases of unusual handwriting.

    • Prediction: MyScript Prediction enables the integration of word completion and next word prediction functionalities into text input applications. It especially targets applications such as input methods and messaging.

    Supported alphabets and languages

    MyScript distinguishes between alphabets and languages.

    Alphabets correspond to symbol sets, scripts. For instance, Latin, or Cyrillic are alphabets. Alphabets do not imply linguistics. MyScript supports the recognition of symbols from various alphabets.

    Languages are necessarily associated to alphabets because a language requires a set of symbols to be written. In addition, languages involve the notion of linguistics. For instance, two languages may use the same set of symbols, i.e. the same alphabet, while having totally different set of words, grammar, etc. For instance, Italian and English uses the Latin scripts. MyScript Text supports the recognition of various languages, e.g. symbols plus language models.
