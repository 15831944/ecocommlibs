MyScript Superimposed

  • Technology
  • Superimposed

    • MyScript Superimposed SDK enables the recognition of letters, words or parts of words written one over the other, without any explicit separation between consecutive fragments (it is also called superimposed handwriting). MyScript Superimposed targets small size devices with sufficient hardware capabilities in terms of CPU power and memory.

    Features

    Languages

    MyScript Superimposed supports 60 languages.

    Alphabets

    MyScript Superimposed supports digits and punctuation marks, Latin, Cyrillic, Greek, Georgian, Arabic and Armenian characters, zhuyin and pinyin transliterations and other symbols.

    Connection with related knowledge

    MyScript Superimposed can identify items within the handwritten text, like dates, phone numbers, or names. It allows building advanced user interactions, such as adding content to the calendar or to contacts, calling a phone number, searching the Web for a name, etc.

    Extending resources recognition power

    MyScript Superimposed allows fine-tuning the recognition process power by creating custom resources. Easy-to-use tools are provided for creating, compiling and managing custom resource files.

    Gesture recognition

    MyScript Superimposed supports gesture recognition: left, right, down then left, down then right gestures.

    Advanced capabilities

    • Text Input Method: MyScript Text Input Method uses multi-threading to recognize first strokes in background, while the user keeps writing more strokes. This enables interactive applications.

    • Prediction: MyScript Prediction is an extension SDK to the MyScript Text handwriting recognition system. It is designed for providing word completion and next word prediction capabilities.

    MyScript Superimposed is not compatible with MyScript WriterAdaptation, nor MyScript InkSearch.

    Supported languages and characters
