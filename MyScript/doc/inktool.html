InkTool

  • InkTool

    • About InkTool

    InkTool is a handy (and free!) Windows application that you can use during your testing phase, in order to:

    • work with a variety of ink file formats from various devices,

    • create .ink files through a graphics tablet or a tablet PC,

    • read stroke input sequences,

    • set up and launch recognition to get an idea of the impact of your settings.

    Provided that you have purchased them, all the MyScript SDKs (except for MyScript Ink, MyScript Gesture and MyScript Document) can be tested within InkTool.

    Please note that the InkTool application is supplied “as is” and does not qualify for any customer support.

    The current version of InkTool does include very useful features, such as a stroke viewer, to get a visual representation of the strokes that make up a particular piece of input. It also has a separate pane under the Recognition tab that gives a detailed recognition result with segmentation and scores. There is a full application help and we recommend that you refer to this for details on the full range of InkTool’s possibilities.

    You will be able to launch InkTool from the tools folder in the root folder (tools/InkTool/bin/win-x64 or win-x86 depending on your system).

    Demonstration

    To see a demonstration recognition process in InkTool, open the application and then the demo file corresponding to the language package you have purchased. The demo files (language_country-prefix-demo.itf) are in the InkTool folder, organized into subfolders per language.

    Once in the demo file, click the Rec button on the InkTool toolbar to run the demonstration. The Edit Layout button shows you all the parameters used to set up recognition.

    There are examples of recognition that you can open in InkTool. Use the “*.itf” files which are in the edk\examples\ink directory. If you open it, you can display the detailed results with the View result details button.
