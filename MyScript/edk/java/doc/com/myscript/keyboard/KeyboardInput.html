com.myscript.keyboard

Class KeyboardInput

  • java.lang.Object
    • com.myscript.engine.EngineObject
      • com.myscript.engine.Input
        • com.myscript.keyboard.KeyboardInput
  • All Implemented Interfaces:
    IDisposable, IEngineObject, IInput, java.lang.Cloneable

    public final class KeyboardInput
extends Input
    The KeyboardInput class models a keyboard input.

    Please note that the implementation of the KeyboardInput class is not synchronized. If multiple threads concurrently try to change the state of a KeyboardInput, explicit synchronization must be done externally.

      • addTransliterationBreak

        public final void addTransliterationBreak()
                                   throws java.lang.IllegalStateException
        Ends the transliteration sequence for the current symbol.

        Note: the next input event will start the transliteration of another symbol.

        Note: this is API is specific to zh_CN stroke transliteration.

        Throws:
        java.lang.IllegalStateException - when this KeyboardInput has been disposed of.
