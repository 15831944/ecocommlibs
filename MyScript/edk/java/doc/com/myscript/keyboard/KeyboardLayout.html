Class KeyboardLayout

  • java.lang.Object
    • com.myscript.engine.EngineObject
      • com.myscript.keyboard.KeyboardLayout
  • All Implemented Interfaces:
    IDisposable, IEngineObject, java.lang.Cloneable

    public final class KeyboardLayout
extends EngineObject
    The KeyboardLayout class models a keyboard layout.

    Please note that the implementation of the KeyboardLayout class is not synchronized. If multiple threads concurrently try to change the state of a KeyboardLayout , explicit synchronization must be done externally.
