com.myscript.shape

Class ShapeDocument

  • java.lang.Object
    • com.myscript.engine.EngineObject
      • com.myscript.shape.ShapeDocument
  • All Implemented Interfaces:
    IAttachTarget, IDisposable, IEngineObject, java.lang.Cloneable

    public final class ShapeDocument
extends EngineObject
implements IAttachTarget
    The ShapeDocument class models the main document.

    Please note that the implementation of the ShapeDocument class is not synchronized. If multiple threads concurrently try to change the state of a ShapeDocument object, explicit synchronization must be done externally.

        public final void commitResult(ShapeInkRange[] ranges)
                        throws java.lang.NullPointerException,
                               java.lang.IllegalArgumentException,
                               java.lang.IllegalStateException
        Marks a portion of this ShapeDocument segments as commited. The Shape recognition results associated with commited portions of a document are used as input of Shape recognition process, rather than the corresponding input strokes. Passing null as the ranges parameter has the same effect as invoking the nullary overload of the method.
        Parameters:
        ranges - the array of ink ranges associated to the portion of ShapeDocument segments to mark as commited.
        Throws:
        java.lang.IllegalStateException - when this ShapeDocument object has been disposed of.
        java.lang.IllegalArgumentException - when ranges < 0.
        java.lang.NullPointerException
