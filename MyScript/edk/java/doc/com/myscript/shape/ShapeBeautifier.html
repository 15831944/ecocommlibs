com.myscript.shape

Class ShapeBeautifier

  • java.lang.Object
    • com.myscript.engine.EngineObject
      • com.myscript.shape.ShapeBeautifier
  • All Implemented Interfaces:
    IContentFieldProcessor, IAttachTarget, IDisposable, IEngineObject, IShapeDocumentProcessor, java.lang.Cloneable

    public final class ShapeBeautifier
extends EngineObject
implements IAttachTarget, IShapeDocumentProcessor, IContentFieldProcessor
    The ShapeBeautifier class models a shape beautifier working on a ShapeDocument. The beautification process consists in connecting and aligning the shapes that have been recognized.

    The ShapeBeautifier class implements the IShapeDocumentProcessor interface. The process method is used to initiate the recognition process.

    Please note that the implementation of the ShapeBeautifier class is not synchronized. If multiple threads concurrently try to change the state of a ShapeBeautifier object, explicit synchronization must be done externally.
