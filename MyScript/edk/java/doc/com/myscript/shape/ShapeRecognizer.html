com.myscript.shape

Class ShapeRecognizer

  • java.lang.Object
    • com.myscript.engine.EngineObject
      • com.myscript.shape.ShapeRecognizer
  • All Implemented Interfaces:
    IContentFieldProcessor, IAttachTarget, IDisposable, IEngineObject, IShapeDocumentProcessor, java.lang.Cloneable

    public final class ShapeRecognizer
extends EngineObject
implements IAttachTarget, IShapeDocumentProcessor, IContentFieldProcessor
    The ShapeRecognizer class models a shape recognizer working on a ShapeDocument.

    The ShapeRecognizer class implements the IShapeDocumentProcessor interface. The process method is used to initiate the recognition process.

    Please note that the implementation of the ShapeRecognizer class is not synchronized. If multiple threads concurrently try to change the state of a ShapeRecognizer object, explicit synchronization must be done externally.
