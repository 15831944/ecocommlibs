Package com.myscript.shape

The com.myscript.shape package provides classes and interfaces that model the access to the MyScript Shape technology.

See: Description

  • Interface Summary 
    Interface Description
    IShapeDocumentProcessor
    The IShapeDocumentProcessor interface models an EngineObject that is capable of processing a target ShapeDocument object.
  • Class Summary 
    Class Description
    DecorationType
    The DecorationType class is a type safe enumeration type that describes the different decoration types of a ShapeDecoratedLine or ShapeDecoratedEllipticArc.
    ShapeBeautifier
    The ShapeBeautifier class models a shape beautifier working on a ShapeDocument.
    ShapeCandidate
    The ShapeCandidate class is the abstract base class for all shape candidates.
    ShapeDecoratedEllipticArc
    The ShapeDecoratedEllipticArc class models a 2D decorated elliptic arc.
    ShapeDecoratedEllipticArcData
    The ShapeDecoratedEllipticArcData class models the data of a decorated elliptic arc in the 2D space.
    ShapeDecoratedLine
    The ShapeDecoratedLine class models a 2D decorated line segment.
    ShapeDecoratedLineData
    The ShapeDecoratedLineData class models the data of a decorated line segment in the 2D space.
    ShapeDocument
    The ShapeDocument class models the main document.
    ShapeEllipticArc
    The ShapeEllipticArc class models a 2D elliptic arc.
    ShapeEllipticArcData
    The ShapeEllipticArcData class models the data of an elliptic arc in the 2D space.
    ShapeErased
    The ShapeErased class models a portion of ink that has been erased by a scratch out.
    ShapeHistoryManager
    The ShapeHistoryManager class models an entity that holds the undo/redo history of a ShapeDocument.
    ShapeInkRange
    The ShapeInkRange class models a range in the input.
    ShapeKnowledge
    The ShapeKnowledge class models the knowledge of the different shapes and their segmentation characteristics for a specific domain.
    ShapeLine

    ShapeSegment

  • Exception Summary 
    Exception Description
    MissingShapeKnowledgeException
    The MissingShapeKnowledgeException exception class models a situation where the MyScript Engine failed to perform the requested operation because a ShapeKnowledge object is missing.
    UnmatchedShapeSegmentException
    The MissingShapeKnowledgeException exception class models a situation where the MyScript Engine failed to perform the requested operation because an endSegment call does not match any prior startSegment call.

Package com.myscript.shape Description

The com.myscript.shape package provides classes and interfaces that model the access to the MyScript Shape technology.

The com.myscript.shape package depends on the com.myscript.engine package for interaction with the native side of MyScript Engine.
