Class ShapeSegment

  • java.lang.Object
    • com.myscript.engine.EngineObject
      • com.myscript.shape.ShapeSegment
  • All Implemented Interfaces:
    IDisposable, IEngineObject, java.lang.Cloneable

    public final class ShapeSegment
extends EngineObject
    The ShapeSegment class models a semantic portion of ink.

    Please note that the implementation of the ShapeSegment class is not synchronized. If multiple threads concurrently try to change the state of a ShapeSegment object, explicit synchronization must be done externally.
