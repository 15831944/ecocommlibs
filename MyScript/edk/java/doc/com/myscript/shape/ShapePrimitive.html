Class ShapePrimitive

  • java.lang.Object

    Direct Known Subclasses:
    ShapeDecoratedEllipticArc, ShapeDecoratedLine, ShapeEllipticArc, ShapeLine

    public abstract class ShapePrimitive
extends EngineObject
    The ShapePrimitive class is the abstract base class that models all the geometric primitives into which a shape model can be decomposed.

    Please note that the implementation of the ShapePrimitive class is not synchronized. If multiple threads concurrently try to change the state of a ShapePrimitive object, explicit synchronization must be done externally.
