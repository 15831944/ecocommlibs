Class ShapeDecoratedLine

    public final class ShapeDecoratedLine
extends ShapePrimitive
    The ShapeDecoratedLine class models a 2D decorated line segment.

    Please note that the implementation of the ShapeDecoratedLine class is not synchronized. If multiple threads concurrently try to change the state of a ShapeDecoratedLine object, explicit synchronization must be done externally.
