Class ShapeModel

    public final class ShapeModel
extends EngineObject
    The ShapeModel class models a kind of shape: triangle, circle, square, etc.

    Please note that the implementation of the ShapeModel class is not synchronized. If multiple threads concurrently try to change the state of a ShapeModel object, explicit synchronization must be done externally.
