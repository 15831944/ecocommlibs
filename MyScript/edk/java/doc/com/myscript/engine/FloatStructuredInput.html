Class FloatStructuredInput

    public final class FloatStructuredInput
extends StructuredInput
implements IFloatStructuredInput
    The FloatStructuredInput class models a structured input to the MyScript technologies where the parameters are expressed in the float number format.

    A FloatStructuredInput object is serializable, in other words you can invoke the load and store methods on it.

    Please note that the implementation of the FloatStructuredInput class is not synchronized. If multiple threads concurrently try to change the state of a FloatStructuredInput, explicit synchronization must be done externally.
