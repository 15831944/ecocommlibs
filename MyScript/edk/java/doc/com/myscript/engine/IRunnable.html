    public interface IRunnable

      • run

        void run(IProgress progress)
         throws java.lang.IllegalStateException,
                CanceledException
        Runs the treatment to be performed by this runnable object.
        Parameters:
        progress - an object implementing the IProgress interface used to notify the client side about the status of the run operation. Passing null as the progress parameter has the same effect as invoking the nullary overload of the method.
        Throws:
        java.lang.IllegalStateException - when this runnable object is not in a state where the run method can be invoked.
        java.lang.IllegalStateException - when this runnable object has been disposed of.
        CanceledException - when the operation has been canceled by the IProgress.update(int, int) method.
