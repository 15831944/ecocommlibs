    public abstract class EngineObject
extends java.lang.Object
implements IEngineObject, java.lang.Cloneable
    The EngineObject class is the abstract base class of all the objects that expose the different aspects of the MyScript technologies.

    Each EngineObject is linked to a specific Engine , which can be viewed as its parent.

    Remember that when you dispose of a parent Engine, all its children are also disposed of.

    Please note that the implementation of the EngineObject class is not synchronized. If multiple threads concurrently try to change the state of an EngineObject, explicit synchronization must be done externally.

        Sets the maximum number of EngineObject objects that can exist at the same time in the user side of a MyScript Engine enabled application.

        The special value zero means no limit. Other values are adjusted to the next multiple of 16. A value under the current count is permitted and will constrain the future EngineObject creations.

        Please note that this limit only applies to EngineObject that do not define their own user limit.

        Parameters:
