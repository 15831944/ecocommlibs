    public abstract class StructuredInput
extends Input
implements IStructuredInput
    The StructuredInput class is the abstract base class of all the MyScript structured input objects.

    A StructuredInput object is serializable, in other words you can invoke the load and store methods on it.

    Please note that the implementation of the StructuredInput class is not synchronized. If multiple threads concurrently try to change the state of a StructuredInput object, explicit synchronization must be done externally.
