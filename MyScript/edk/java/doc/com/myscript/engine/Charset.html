Class Charset

    • com.myscript.engine.EngineObject

  • All Implemented Interfaces:
    IDisposable, IEngineObject, java.lang.Cloneable

    public final class Charset
extends EngineObject
    The Charset class models the MyScript Engine Charset native type.

    Charset objects are responsible for the translation between character strings and the MyScript Engine internal characters.

    Please note that the implementation of the Charset class is not synchronized. If multiple threads concurrently try to change the state of a Charset object, explicit synchronization must be done externally.
