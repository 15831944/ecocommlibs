      • getAutoPurgeFactor

        public final float getAutoPurgeFactor()
                               throws java.lang.IllegalStateException
        Returns the factor applied to the maximum depth of the undo stack to compute the threshold above which automatic purge will be triggered. The default value is 1.5 which means the automatic purge will be triggered when the current undo stack depth goes above 75, if the maximum depth of the undo stack is the default value: 50. Automatic purge will then reduce the undo stack back to the maximum depth.
        Returns:
        the value of the automatic purge factor.
        Throws:
        java.lang.IllegalStateException - when this HistoryManager has been disposed of.
