com.myscript.engine

Class Version

  • java.lang.Object
    • com.myscript.engine.Version

  • public final class Version
extends java.lang.Object
    The Version class models a specific version of an instance of the Engine class.

    A version number is made of three components:

    • a 7 bit major version number
    • a 7 bit minor version number
    • a 15 bit revision version number
    Since:
    4.0.0
    Version:
    1.0
