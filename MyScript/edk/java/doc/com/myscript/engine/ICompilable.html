Interface ICompilable

  • All Superinterfaces:
    IDisposable, IEngineObject
    All Known Implementing Classes:
    LexicalUnit, Lexicon

    public interface ICompilable
extends IEngineObject
    The ICompilable interface models an EngineObject that is compilable, in other words on which you can invoke the compile and sometimes decompile methods.

      • compile

        void compile(IProgress progress)
             throws java.lang.IllegalStateException,
                    CanceledException
        Compiles this compilable object.
        Parameters:
        progress - an object implementing the IProgress interface used to notify the client side about the status of the compile operation. Passing null as the progress parameter has the same effect as invoking the nullary overload of the method.
        Throws:
        java.lang.IllegalStateException - when this compilable object has been disposed of.
        java.lang.IllegalStateException - when this compilable object is not in a state where it can be compiled: either the object has not been edited or it is already compiled.
        CanceledException - when the operation has been canceled by the IProgress.update(int, int) method.
