    public abstract class Iterator
extends EngineObject
    The Iterator class models an iterator over a MyScript Engine collection.

    Please note that the implementation of the Iterator class is not synchronized. If multiple threads concurrently try to change the state of an Iterator object, explicit synchronization must be done externally.
