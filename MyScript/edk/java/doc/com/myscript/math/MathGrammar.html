com.myscript.math

Class MathGrammar

  • java.lang.Object
    • com.myscript.engine.EngineObject
      • com.myscript.math.MathGrammar
  • All Implemented Interfaces:
    IDisposable, IEngineObject, java.lang.Cloneable

    public final class MathGrammar
extends EngineObject
    The MathGrammar class models a grammar used to recognize maths.

    Please note that the implementation of the MathGrammar class is not synchronized. If multiple threads concurrently try to change the state of an MathGrammar object, explicit synchronization must be done externally.
