Class MathNode

  • java.lang.Object

    Direct Known Subclasses:
    MathNonTerminalNode, MathRuleNode, MathTerminalNode

    public abstract class MathNode
extends EngineObject
    The MathNode class is the abstract base class that models all the nodes of an math tree.

    Please note that the implementation of the MathNode class is not synchronized. If multiple threads concurrently try to change the state of a MathNode object, explicit synchronization must be done externally.

        public static final void setUserLimit(Engine engine,

                               throws java.lang.NullPointerException,
                                      java.lang.IllegalStateException,
                                      java.lang.IllegalArgumentException
        Sets the maximum number of MathNode objects that can exist at the same time in the user side of a MyScript Engine enabled application.

        The special value zero means no limit. Other values are adjusted to the next multiple of 16. A value under the current count is permitted and will constrain the future MathNode object creations.

        Parameters:
        engine - an Engine object.
        limit - the value of the new creation limit.
        Throws:
        java.lang.NullPointerException - when engine is null.
        java.lang.IllegalStateException - when engine has been disposed of.
        java.lang.IllegalArgumentException - when limit < 0.
