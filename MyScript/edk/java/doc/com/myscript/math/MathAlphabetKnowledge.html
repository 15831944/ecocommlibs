Class MathAlphabetKnowledge

    public final class MathAlphabetKnowledge
extends EngineObject
implements IMathAlphabet
    The MathAlphabetKnowledge class models the list of characters and mathematical symbols that can be written in an math.

    Please note that the implementation of the MathAlphabetKnowledge class is not synchronized. If multiple threads concurrently try to change the state of an MathAlphabetKnowledge object, explicit synchronization must be done externally.
