com.myscript.math

Class MathRecognizer

  • java.lang.Object
    • com.myscript.engine.EngineObject
      • com.myscript.math.MathRecognizer
  • All Implemented Interfaces:
    IContentFieldProcessor, IAttachTarget, IDisposable, IEngineObject, java.lang.Cloneable

    public final class MathRecognizer
extends EngineObject
implements IAttachTarget, IContentFieldProcessor
    The MathRecognizer class models an math recognizer.

    Please note that the implementation of the MathRecognizer class is not synchronized. If multiple threads concurrently try to change the state of an MathRecognizer object, explicit synchronization must be done externally.
