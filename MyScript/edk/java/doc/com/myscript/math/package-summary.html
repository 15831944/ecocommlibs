Package com.myscript.math

The com.myscript.math package provides classes and interfaces that model the access to the MyScript Math technology.

Package com.myscript.math Description

The com.myscript.math package provides classes and interfaces that model the access to the MyScript Math technology.

The com.myscript.math package depends on the com.myscript.engine package for interaction with the native side of MyScript Engine.
