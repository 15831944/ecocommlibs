com.myscript.math

Class MathDocument

  • java.lang.Object
    • com.myscript.engine.EngineObject
      • com.myscript.math.MathDocument
  • All Implemented Interfaces:
    IDisposable, IEngineObject, java.lang.Cloneable

    public final class MathDocument
extends EngineObject
    The MathDocument class models the main math document.

    Please note that the implementation of the MathDocument class is not synchronized. If multiple threads concurrently try to change the state of a MathDocument object, explicit synchronization must be done externally.
