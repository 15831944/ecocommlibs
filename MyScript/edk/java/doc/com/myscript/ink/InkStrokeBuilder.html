        public long getTimestamp()
        Returns the timestamp of the stroke as the number of microseconds elapsed since the Unix EPOCH (January 1, 1970, 00:00:00 GMT).

        Note that you will generally have to convert the value to milliseconds if you use it in other Java timing utility. For example, if you want to print the value you could use: 

         String t = new java.sql.Timestamp(stroke.getTimestamp() / 1000).toString();
