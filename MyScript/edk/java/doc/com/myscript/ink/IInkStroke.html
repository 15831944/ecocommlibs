      • getTimestamp

        long getTimestamp()
        Returns the timestamp of the stroke as the number of microseconds elapsed since the Unix EPOCH (January 1, 1970, 00:00:00 GMT).

        Note that you will generally have to convert the value to milliseconds if you use it in other Java timing utility. For example, if you want to print the value you could use: 

         String t = new java.sql.Timestamp(stroke.getTimestamp() / 1000).toString();
        Returns:
        the timestamp of the stroke.

      • getPoints

        void getPoints(int sampleOffset,
             int sampleCount,
             float[] xValues,
             int xOffset,
             int xStride,
             float[] yValues,
             int yOffset,
             int yStride)
        Returns the float values of the "X" and "Y" channels of the stroke. Note that you can use a single interleaved array by passing in suitable offsets and strides, for example: 
         float[] xy = new float[20];
 stroke.getPoints(0, 20, xy, 0, 2, xy, 1, 2);
        Parameters:
        sampleOffset - the offset of the points to retrieve.
        sampleCount - the number of points to retrieve.
        xValues - the array to be filled with the values of the "X" channel.
        xOffset - the index of the first fill position.
        xStride - the increment between 2 fill positions.
        yValues - the array to be filled with the values of the "Y" channel.
        yOffset - the index of the first fill position.
        yStride - the increment between 2 fill positions.
