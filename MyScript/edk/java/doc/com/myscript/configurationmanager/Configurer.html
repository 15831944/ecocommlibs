Class Configurer

  • java.lang.Object
    • com.myscript.engine.EngineObject
      • com.myscript.configurationmanager.Configurer
  • All Implemented Interfaces:
    IDisposable, IEngineObject, java.lang.Cloneable

    public final class Configurer
extends EngineObject
    The Configurer class models a configurer.

    Please note that the implementation of the Configurer class is not synchronized. If multiple threads concurrently try to change the state of a Configurer object, explicit synchronization must be done externally.
