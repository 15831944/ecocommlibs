Package com.myscript.analyzer

The com.myscript.analyzer package provides classes and interfaces that model the access to the MyScript Analyzer technology.

See: Description

  • Interface Summary 
    Interface Description
    IAnalyzerDocumentProcessor
    The IAnalyzerDocumentProcessor interface models an EngineObject that is capable of processing a target AnalyzerDocument object.
  • Class Summary 
    Class Description
    Analyzer
    The Analyzer class models an analyzer working on an AnalyzerDocument.
    AnalyzerCell
    The AnalyzerCell class models a cell.
    AnalyzerCellData
    The AnalyzerCellData class models the range of a cell space.
    AnalyzerDocument
    The AnalyzerDocument class models a document.
    AnalyzerElement
    The AnalyzerElement class is the abstract base class that models all the analyzer elements into which an analyzer document can be decomposed.
    AnalyzerGroup
    The AnalyzerGroup class models a group.
    AnalyzerGroupType
    The AnalyzerGroupType class is a type safe enumeration type that describes the different types of an AnalyzerGroup.
    AnalyzerInkRange
    The AnalyzerInkRange class models a range in the input.
    AnalyzerJustificationType
    The AnalyzerJustificationType class is a type safe enumeration type that describes the different types of justification of an AnalyzerTextLineData.
    AnalyzerKnowledge
    The AnalyzerKnowledge class models the knowledge of the different analyzer elements and their segmentation characteristics for a specific domain.
    AnalyzerLine
    The AnalyzerLine class models a line.
    AnalyzerLineData
    The AnalyzerLineData class models the data of a line segment in the 2D space.
    AnalyzerPointData
    The AnalyzerPointData class models the location of a point in the 2D space.
    AnalyzerProcessingLevel
    The AnalyzerProcessingLevel class is a type safe enumeration type that describes the different processing levels available.
    AnalyzerShape
    The AnalyzerShape class models a shape.
    AnalyzerStrokeType
    The AnalyzerStrokeType class is a type safe enumeration type that describes the different types of a stroke.
    AnalyzerTable
    The AnalyzerTable class models a table.
    AnalyzerTableData
    The AnalyzerTableData class models the data of a table.
    AnalyzerTextLine
    The AnalyzerTextLine class models a text line.
    AnalyzerTextLineData
    The AnalyzerTextLineData class models the data of a text line.
    AnalyzerUnderline
    The AnalyzerUnderline class models an underline element.
    AnalyzerUnderlineData
    The AnalyzerUnderlineData class models the data of an underline.
  • Exception Summary 
    Exception Description
    MissingAnalyzerKnowledgeException
    The MissingAnalyzerKnowledgeException exception class models a situation where the MyScript Engine failed to perform the requested operation because an AnalyzerKnowledge object is missing.

Package com.myscript.analyzer Description

The com.myscript.analyzer package provides classes and interfaces that model the access to the MyScript Analyzer technology.

The com.myscript.analyzer package depends on the com.myscript.engine package for interaction with the native side of MyScript Engine.
