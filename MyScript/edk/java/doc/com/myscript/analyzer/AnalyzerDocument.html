com.myscript.analyzer

Class AnalyzerDocument

  • java.lang.Object
    • com.myscript.engine.EngineObject
      • com.myscript.analyzer.AnalyzerElement
        • com.myscript.analyzer.AnalyzerDocument
  • All Implemented Interfaces:
    IDisposable, IEngineObject, java.lang.Cloneable

    public final class AnalyzerDocument
extends AnalyzerElement
    The AnalyzerDocument class models a document.

    Please note that the implementation of the AnalyzerDocument class is not synchronized. If multiple threads concurrently try to change the state of a AnalyzerDocument object, explicit synchronization must be done externally.

        public final void setCoordinateResolution(float resolution)
                                   throws java.lang.IllegalStateException,
                                          java.lang.IllegalArgumentException
        Sets the resolution of the coordinates in dpi. This is the resolution of the grid of integer values in the chosen coordinate system. For example, if you use floating-point pixel coordinates you must give the screen resolution, no matter the resolution of the capturing device.
        Parameters:
        resolution - the coordinate resolution.
        Throws:

        java.lang.IllegalArgumentException - when resolution is infinitely large in magnitude.
