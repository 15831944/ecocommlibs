Class AnalyzerElement

  • java.lang.Object
    • com.myscript.engine.EngineObject
      • com.myscript.analyzer.AnalyzerElement
  • All Implemented Interfaces:

    Direct Known Subclasses:
    AnalyzerCell, AnalyzerDocument, AnalyzerGroup, AnalyzerLine, AnalyzerShape, AnalyzerTable, AnalyzerTextLine, AnalyzerUnderline

    public abstract class AnalyzerElement
extends EngineObject
    The AnalyzerElement class is the abstract base class that models all the analyzer elements into which an analyzer document can be decomposed.

    Please note that the implementation of the AnalyzerElement class is not synchronized. If multiple threads concurrently try to change the state of a AnalyzerElement object, explicit synchronization must be done externally.

        public static final void setUserLimit(Engine engine,

                               throws java.lang.NullPointerException,
                                      java.lang.IllegalStateException,
                                      java.lang.IllegalArgumentException
        Sets the maximum number of AnalyzerElement objects that can exist at the same time in the user side of a MyScript Engine enabled application.

        The special value zero means no limit. Other values are adjusted to the next multiple of 16. A value under the current count is permitted and will constrain future AnalyzerElement object creations.

        Parameters:
        engine - an Engine object.
        limit - the value of the new creation limit.
        Throws:
        java.lang.NullPointerException - when engine is null.
        java.lang.IllegalStateException - when engine has been disposed of.
        java.lang.IllegalArgumentException - when limit < 0.
