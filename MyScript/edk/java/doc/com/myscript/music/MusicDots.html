    public final class MusicDots
extends MusicElement
    The MusicDots class models dots written beside a note or a rest in a MusicDocument.

    Please note that the implementation of the MusicDots class is not synchronized. If multiple threads concurrently try to change the state of a MusicDots object, explicit synchronization must be done externally.
