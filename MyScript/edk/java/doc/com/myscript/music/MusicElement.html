    public abstract class MusicElement
extends EngineObject
    The MusicElement is the abstract base class that models the different elements in a recognized MusicDocument.

    Please note that the implementation of the MusicElement class is not synchronized. If multiple threads concurrently try to change the state of a MusicElement object, explicit synchronization must be done externally.

                                      java.lang.IllegalStateException,

        Sets the maximum number of MusicElement objects that can exist at the same time in the user side of a MyScript Engine enabled application.

        The special value zero means no limit. Other values are adjusted to the next multiple of 16. A value under the current count is permitted and will constrain the future MusicElement object creations.

        Parameters:
        engine - an Engine object.
        limit - the value of the new creation limit.
        Throws:
        java.lang.NullPointerException - when engine is null.
        java.lang.IllegalStateException - when engine has been disposed of.
        java.lang.IllegalArgumentException - when limit < 0.
