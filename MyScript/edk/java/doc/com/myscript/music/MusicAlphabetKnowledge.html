    public final class MusicAlphabetKnowledge
extends EngineObject
    The MusicAlphabetKnowledge class models the list of musical symbols that can be written in a music document.

    Please note that the implementation of the MusicAlphabetKnowledge class is not synchronized. If multiple threads concurrently try to change the state of a MusicAlphabetKnowledge object, explicit synchronization must be done externally.
