Class MusicRest

  • java.lang.Object
    • com.myscript.engine.EngineObject
      • com.myscript.music.MusicElement
        • com.myscript.music.MusicRest
  • All Implemented Interfaces:
    IDisposable, IEngineObject, java.lang.Cloneable

    public final class MusicRest
extends MusicElement
    The MusicRest class models an rest written in a MusicDocument.

    Please rest that the implementation of the MusicRest class is not synchronized. If multiple threads concurrently try to change the state of a MusicRest object, explicit synchronization must be done externally.
