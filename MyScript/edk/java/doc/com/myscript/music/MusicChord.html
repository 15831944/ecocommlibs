com.myscript.music

Class MusicChord

  • java.lang.Object
    • com.myscript.engine.EngineObject
      • com.myscript.music.MusicElement
        • com.myscript.music.MusicChord
  • All Implemented Interfaces:
    IDisposable, IEngineObject, java.lang.Cloneable

    public final class MusicChord
extends MusicElement
    The MusicChord class models an chord written in a MusicDocument.

    Please chord that the implementation of the MusicChord class is not synchronized. If multiple threads concurrently try to change the state of a MusicChord object, explicit synchronization must be done externally.
