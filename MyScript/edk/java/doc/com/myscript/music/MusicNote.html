com.myscript.music

Class MusicNote

  • java.lang.Object
    • com.myscript.engine.EngineObject
      • com.myscript.music.MusicElement
        • com.myscript.music.MusicNote
  • All Implemented Interfaces:
    IDisposable, IEngineObject, java.lang.Cloneable

    public final class MusicNote
extends MusicElement
    The MusicNote class models an note written in a MusicDocument.

    Please note that the implementation of the MusicNote class is not synchronized. If multiple threads concurrently try to change the state of a MusicNote object, explicit synchronization must be done externally.
