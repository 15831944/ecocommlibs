com.myscript.music

Class MusicDocument

  • java.lang.Object
    • com.myscript.engine.EngineObject
      • com.myscript.music.MusicDocument
  • All Implemented Interfaces:
    IDisposable, IEngineObject, java.lang.Cloneable

    public final class MusicDocument
extends EngineObject
    The MusicDocument class models the main music document.

    Please note that the implementation of the MusicDocument class is not synchronized. If multiple threads concurrently try to change the state of a MusicDocument object, explicit synchronization must be done externally.
