      • getTypesetData

        public final TypesetData getTypesetData(ContentField field,
                         PageSelection selection,
                         ContentProcessor contentProcessor)
        Returns the typeset data corresponding to selected area as a TypesetData object. This typeset data can then be modified and finally applied to the document. Passing null as the contentProcessor parameter has the same effect as invoking the nullary overload of the method.
        Parameters:
        field - the associated content field.
        selection - the portion of the page to generate typeset from, as a PageSelection.
        contentProcessor - the content processor used to run recognition on field.
        Returns:
        a typeset data describing the outcome of the typeset operation as a TypesetData object.
        Throws:
        java.lang.IllegalStateException - when the content of selection has been modified since recognition results in field were updated.
        java.lang.NullPointerException - when field is null.
        java.lang.NullPointerException - when selection is null.
        Since:
        7.0.0
