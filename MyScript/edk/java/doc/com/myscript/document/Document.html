      • removeObject

        public final void removeObject(java.lang.String url)
        Removes an object from the document. The object will not be physically removed from the document file immediately, to allow undo/redo of this action. However, when the document is released this object will not be accessible anymore and next time the document is loaded, object will be physically deleted.
        Parameters:
        url - the URL of the object within the document.
