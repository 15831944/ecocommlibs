com.myscript.document

Class TextInspector

  • java.lang.Object
    • com.myscript.engine.EngineObject
      • com.myscript.document.TextInspector
  • All Implemented Interfaces:
    IDisposable, IEngineObject, java.lang.Cloneable

    public class TextInspector
extends EngineObject
    The TextInspector class models a tool meant to make it easier to access, select and manipulate the text recognition result associated with a page.

    Please note that the implementation of the TextInspector class is not synchronized. If multiple threads concurrently try to change the state of a TextInspector object, explicit synchronization must be done externally.
