Class ContentProcessorListener

  • All Implemented Interfaces:
    IDisposable, IEngineObject, java.lang.Cloneable

    public class ContentProcessorListener
extends UserObject
    The ContentProcessorListener is the abstract base class that models an object that will receive notification of changes that occur on an ContentProcessor object.

    Please note that the implementation of the ContentProcessorListener class is not synchronized. If multiple threads concurrently try to change the state of a ContentProcessorListener object, explicit synchronization must be done externally.
