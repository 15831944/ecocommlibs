com.myscript.document

Class TypesetItem

  • java.lang.Object
    • com.myscript.engine.EngineObject
      • com.myscript.document.TypesetItem
  • All Implemented Interfaces:
    IDisposable, IEngineObject, java.lang.Cloneable

    public final class TypesetItem
extends EngineObject
    The TypesetItem class models a glyph or primitive generated from recognition results, that will replace a given portion of ink when the associated typeset data is applied.

    Please note that the implementation of the TypesetItem class is not synchronized. If multiple threads concurrently try to change the state of a TypesetItem object, explicit synchronization must be done externally.
