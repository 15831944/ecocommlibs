Package com.myscript.prediction Description

The com.myscript.prediction package provides classes and interfaces that model the access to the MyScript prediction technology.

The com.myscript.prediction package depends on the com.myscript.engine package for the interaction with the native side of MyScript Engine and on the com.myscript.text package that allows access to the MyScript Handwriting Recognition technology.
