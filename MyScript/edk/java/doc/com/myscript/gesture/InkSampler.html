Class InkSampler

  • java.lang.Object
    • com.myscript.engine.EngineObject
      • com.myscript.gesture.InkSampler
  • All Implemented Interfaces:
    IDisposable, IEngineObject, IInkStrokeSet, java.lang.Cloneable

    public final class InkSampler
extends EngineObject
implements IInkStrokeSet
    The InkSampler is the abstract base class that models an ink sampler.

    Please note that the implementation of the InkSampler class is not synchronized. If multiple threads concurrently try to change the state of a InkSampler object, explicit synchronization must be done externally.

        public void pointerCancel(long deviceId,
                 long pointerId)
                   throws java.lang.IllegalArgumentException,
                          java.lang.NullPointerException,
                          java.lang.IllegalStateException,
                          NotPermittedFromThisThreadException
        Cancels an ongoing pointer trace in this InkSampler. What we call an ongoing pointer trace here is a sequence of pointer events starting with a pointerDown(long, long, Object[]) or pointerDown(long, long, float[]) and followed by zero or more pointerMove(long, long, Object[]) or pointerMove(long, long, float[]) events with the same deviceId and pointerId.
        Parameters:
        deviceId - the id of the device.
        pointerId - the id of the pointer.
        Throws:
        java.lang.IllegalArgumentException - when deviceId is not registered.
        java.lang.IllegalArgumentException - when pointerId is not registered.
        java.lang.IllegalStateException - when InkSampler has been disposed of.
        NotPermittedFromThisThreadException - the pointerCancel(long, long) call was made from a different thread from the one this InkSampler was created in.
        java.lang.NullPointerException
