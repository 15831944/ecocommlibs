Class InkSamplerListener

  • All Implemented Interfaces:
    IDisposable, IEngineObject, java.lang.Cloneable

    public class InkSamplerListener
extends UserObject
    The InkSamplerListener is the abstract base class that models an object that will receive notification of changes that occur on an InkSampler object.

    Please note that the implementation of the InkSamplerListener class is not synchronized. If multiple threads concurrently try to change the state of a InkSamplerListener object, explicit synchronization must be done externally.
