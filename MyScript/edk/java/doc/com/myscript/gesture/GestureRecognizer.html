com.myscript.gesture

Class GestureRecognizer

  • java.lang.Object
    • com.myscript.engine.EngineObject
      • com.myscript.gesture.GestureRecognizer
  • All Implemented Interfaces:
    IDisposable, IEngineObject, java.lang.Cloneable

    public final class GestureRecognizer
extends EngineObject
    The GestureRecognizer is the abstract base class that models a gesture recognizer.

    Please note that the implementation of the GestureRecognizer class is not synchronized. If multiple threads concurrently try to change the state of a GestureRecognizer object, explicit synchronization must be done externally.
