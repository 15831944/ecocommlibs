Class GestureResult

  • java.lang.Object
    • com.myscript.engine.EngineObject
      • com.myscript.gesture.GestureResult
  • All Implemented Interfaces:
    IDisposable, IEngineObject, java.lang.Cloneable

    public final class GestureResult
extends EngineObject
    The GestureResult models a gesture result.

    Please note that the implementation of the GestureResult class is not synchronized. If multiple threads concurrently try to change the state of a GestureResult object, explicit synchronization must be done externally.
