    • Field Detail

      • ADD

        public static final GestureIntent ADD
        The intent is to add content. The application should add the associated strokes to the target model. The associated strokes are given by GestureResult.getFirstStroke() and GestureResult.getLastStroke(). Please note that you have to detach the strokes from the InkSampler before to add them to the target model.

      • ERASE

        public static final GestureIntent ERASE
        The intent is to erase content. The application should erase the associated selection.

      • SELECT

        public static final GestureIntent SELECT
        The intent is to select content. The application should update its current selection with the associated selection (e.g. using ISelection.combine(ISelection, SelectionModifier)).

      • INSERT

        public static final GestureIntent INSERT
        The intent is to insert space. The application should translate the after selection in order to make room between it and the before selection. The amount of space to add as well as its direction depend on the gesture type. If the insert gesture is on a typeset space it will be returned into the associated selection .

      • JOIN

        public static final GestureIntent JOIN
        The intent is to join two items from the content (e.g.: two words). The application should translate the after selection in order to remove the space between it and the before selection. Additionally, the associated selection may be non-empty, in which case it should be erased (e.g.: if you scratch out a typeset space).

      • DECORATE

        public static final GestureIntent DECORATE
        The intent is to decorate content. The actual type of decoration is given by the gesture type (e.g. underline).

      • TAP

        public static final GestureIntent TAP
        The intent is to notify for tap events. The actual type of tap is given by the gesture type (e.g. tap).
