com.myscript.text

Class TextSegmenter

  • java.lang.Object
    • com.myscript.engine.EngineObject
      • com.myscript.text.TextSegmenter
  • All Implemented Interfaces:
    IAttachTarget, IDisposable, IEngineObject, java.lang.Cloneable

    public final class TextSegmenter
extends EngineObject
implements IAttachTarget
    The TextSegmenter class models a text segmenter that can split sentences into words.

    Please note that the implementation of the TextSegmenter class is not synchronized. If multiple threads concurrently try to change the state of a TextSegmenter object, explicit synchronization must be done externally.
