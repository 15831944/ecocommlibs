Package com.myscript.text

The com.myscript.text package provides classes and interfaces that model the access to the MyScript Handwriting Recognition technology.

See: Description

  • Interface Summary 
    Interface Description
    IAlphabet
    The IAlphabet interface models an EngineObject object that contains symbols.
    IRecognitionResultProvider
    The IRecognitionResultProvider interface models an EngineObject object that is able to provide a recognition result, in other words on which you can invoke the getResult method.
  • Class Summary 
    Class Description
    AccentuationScheme
    The AccentuationScheme class is a type safe enumeration type that describes the different accentuation schemes of a Vocabulary.
    AlphabetKnowledge
    The AlphabetKnowledge class is the abstract base class that models knowledge of the alphabet and the segmentation characteristics of a specific language.
    Archive
    The Archive class models the MyScript Handwriting Recognition Archive resource type.
    CandidateFlags
    The CandidateFlags type is a type safe bit flags that lists the recognition result candidate flags.
    CandidateIterator
    The CandidateIterator class models an iterator over the candidates of a RecognitionResult.
    CasingScheme
    The CasingScheme class is a type safe enumeration type that describes the different casing schemes of a Vocabulary.
    CursiveAlphabetKnowledge
    The CursiveAlphabetKnowledge class models the MyScript Handwriting Recognition Cursive Alphabet Knowledge resource type.
    DigitalInkTagger
    The DigitalInkTagger class models an entity capable of annotating pieces of digital ink with semantic tags.
    DigitalTextMatcher
    The DigitalTextMatcher class models an entity that is able to tell if some digital text can be recognized by a specified MyScript Handwriting Recognition resource configuration.
    FuzzyPinyinFlags
    The FuzzyPinyinFlags type represents type safe bit flags that configure the different fuzzy pinyin behaviors.
    Grammar
    The Grammar class models the MyScript Handwriting Recognition Grammar resource type.
    HalfWidthFullWidthScheme
    The HalfWidthFullWidthScheme class is a type safe enumeration type that describes the different half width / full width character conversion schemes supported by a Recognizer.
    HandPrintedAlphabetKnowledge
    The HandPrintedAlphabetKnowledge class models the MyScript Handwriting Recognition Hand Printed Alphabet Knowledge resource type.
    HandwritingContext
    The HandwritingContext class models a MyScript technology entity that is able to deal with handwriting.
    InputItemLocator
    The InputItemLocator models the location of an item in the input.
    InputRange
    The InputRange class models a discrete range in the input, denoted by the list of the contiguous range elements that compose it.
    InputRangeElement
    The InputRangeElement models a contiguous range in the input delimited by its first and last enclosed items.
    IsolatedAlphabetKnowledge
    The IsolatedAlphabetKnowledge class models the MyScript Handwriting Recognition Isolated Alphabet Knowledge resource type.
    LexicalUnit
    The LexicalUnit class models the MyScript Handwriting Recognition Lexical Unit resource type.
    Lexicon
    The Lexicon class models the MyScript Handwriting Recognition Lexicon resource type.
    LinguisticKnowledge
    The LinguisticKnowledge class is the abstract base class that models knowledge about the words, phrases or particular semantic units that may exist in a specific language.
    LinguisticKnowledgeBuilder
    The LinguisticKnowledgeBuilder class models an object capable of building a Resource object containing linguistic knowledge derived from user input.
    RecognitionResult
    The RecognitionResult class models the result of a handwriting recognition treatment.
    Recognizer
    The Recognizer class is the abstract base class of all the recognizers provided by the MyScript Handwriting Recognition technology.
    Resource
    The Resource class is the abstract base class that models knowledge about the various language elements you are recognizing.
    SegmentIterator
    The SegmentIterator class models an iterator over the segments; segments are the pieces of input that make up the segmentation of a recognition result candidate.
    SpeedQualityCompromise
    The SpeedQualityCompromise class is a type safe enumeration type that specifies the different compromise level used to control the balance between speed and quality.
    StructuredInputRecognizer
    The StructuredInputRecognizer class models a MyScript handwriting recognizer fueled by a structured input.
    SubsetKnowledge
    The SubsetKnowledge class models the MyScript Handwriting Recognition Subset Knowledge resource type.
    TagIterator
    The TagIterator class models an iterator over tags; tags categorize pieces of input: "this piece of ink corresponds to a text line, this other one corresponds to an erasure, etc".
    TagList
    The TagList class models a list of tags.
    TagType
    The TagType class is a type safe enumeration type that describes the different type of tags in the digital ink.
    TextRecognizer
    The TextRecognizer class models a recognizer that processes MyScript document content fields instead of the legacy input.
    TextSegmenter
    The TextSegmenter class models a text segmenter that can split sentences into words.
    Vocabulary
    The Vocabulary class is the abstract base class that models vocabulary level linguistic knowledge given to the recognizer.
    WordIterator
    The WordIterator class models an iterator over the words of a Lexicon object.
  • Exception Summary 
    Exception Description
    CharStrokeLimitExceededException
    The ChatStrokeLimitExceededException exception class models a situation where the MyScript Engine failed to perform the requested operation because the input contains a character input unit that received more strokes than the limit fixed by the recognizer (please note that this limit may change from one recognizer to another).
    InputLimitExceededException
    The InputLimitExceededException exception class models a situation where the MyScript Engine failed to perform the requested operation because an input with too many strokes or too many input units was sent to the target handwriting context.
    MissingAlphabetKnowledgeException
    The MissingAlphabetKnowledgeException exception class models a situation where the MyScript Engine failed to perform the requested operation because an AlphabetKnowledge resource is missing.
    MissingSourceException
    The MissingSourceException exception class models a situation where the MyScript Engine failed to perform the requested operation because the source to an input consumer object is missing.
    RegexInvalidCharacterException
    The RegexInvalidCharacterException exception class models a parse error in a regular expression: invalid character, which means that an invalid character was found (a character that has no mapping).
    RegexInvalidReferenceException
    The RegexInvalidReferenceException exception class models a parse error in a regular expression: invalid reference, which means that a reference was found but the corresponding definition does not exist.
    RegexSubExpImbalanceException
    The RegexSubExpImbalanceException exception class models a parse error in a regular expression: sub-expression imbalance, which means that the number of open and close parentheses does not match.
    RegexTrailingEscapeException
    The RegexTrailingEscapeException exception class models a parse error in a regular expression: trailing escape, which means that a back slash was found at the end of the expression (without the escaped character).
    RegexUnescapedOperatorException
    The RegexUnescapedOperatorException exception class models a parse error in a regular expression: unescaped operator.
    RegexUnmatchedCloseRefException
    The RegexUnmatchedCloseRefException exception class models a parse error in a regular expression: unmatched open reference, which means that a close brace exists without the matching open brace.
    RegexUnmatchedCloseSetException
    The RegexUnmatchedCloseSetException exception class models a parse error in a regular expression: unmatched close set, which means that a close square bracket exists without the matching open square bracket.
    RegexUnmatchedOpenRefException
    The RegexUnmatchedOpenRefException exception class models a parse error in a regular expression: unmatched open reference, which means that an open brace exists without the matching close brace.
    RegexUnmatchedOpenSetException
    The RegexUnmatchedOpenSetException exception class models a parse error in a regular expression: unmatched open set, which means that an open square bracket exists without the matching close square bracket.

Package com.myscript.text Description

The com.myscript.text package provides classes and interfaces that model the access to the MyScript Handwriting Recognition technology.

The com.myscript.text package depends on the com.myscript.engine package for interaction with the native side of MyScript Engine.
