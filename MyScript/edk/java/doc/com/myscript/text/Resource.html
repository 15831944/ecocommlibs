Class Resource

  • java.lang.Object
    • com.myscript.engine.EngineObject

  • All Implemented Interfaces:
    IDisposable, IEngineObject, java.lang.Cloneable
    Direct Known Subclasses:
    AlphabetKnowledge, Archive, LinguisticKnowledge, SubsetKnowledge

    public abstract class Resource
extends EngineObject
    The Resource class is the abstract base class that models knowledge about the various language elements you are recognizing.

    Please note that the implementation of the Resource class is not synchronized. If multiple threads concurrently try to change the state of a Resource object, explicit synchronization must be done externally. However, it is thread safe to attach the same Resource object to multiple IAttachTarget implementers.

        Sets the maximum number of Resource objects that can exist at the same time in the user side of a MyScript Engine enabled application.

        The special value zero means no limit. Other values are adjusted to the next multiple of 16. A value under the current count is permitted and will constrain future Resource object creations.

        Parameters:
        engine - an Engine object.
        limit - the value of the new creation limit.
