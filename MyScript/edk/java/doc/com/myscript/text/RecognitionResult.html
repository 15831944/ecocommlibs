Class RecognitionResult

  • java.lang.Object
    • com.myscript.engine.EngineObject
      • com.myscript.text.RecognitionResult
  • All Implemented Interfaces:
    IDisposable, IEngineObject, java.lang.Cloneable

    public final class RecognitionResult
extends EngineObject
    The RecognitionResult class models the result of a handwriting recognition treatment.

    Please note that the implementation of the RecognitionResult class is not synchronized. If multiple threads concurrently try to change the state of a RecognitionResult object, explicit synchronization must be done externally.
