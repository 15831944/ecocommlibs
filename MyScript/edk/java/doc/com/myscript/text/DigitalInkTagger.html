Class DigitalInkTagger

  • java.lang.Object

    public final class DigitalInkTagger
extends HandwritingContext
implements IRunnable
    The DigitalInkTagger class models an entity capable of annotating pieces of digital ink with semantic tags.

    Please note that the implementation of the DigitalInkTagger class is not synchronized. If multiple threads concurrently try to change the state of a DigitalInkTagger object, explicit synchronization must be done externally.
