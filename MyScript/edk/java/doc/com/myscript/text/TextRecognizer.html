Class TextRecognizer

  • java.lang.Object
    • com.myscript.engine.EngineObject
      • com.myscript.text.TextRecognizer
  • All Implemented Interfaces:
    IContentFieldProcessor, IAttachTarget, IDisposable, IEngineObject, java.lang.Cloneable

    public final class TextRecognizer
extends EngineObject
implements IAttachTarget, IContentFieldProcessor
    The TextRecognizer class models a recognizer that processes MyScript document content fields instead of the legacy input. A TextRecognizer is configured just like any other TextRecognizer, with the same properties and attachments. After it is configured you can use a ContentFieldProcessorAdapter to process content fields.
