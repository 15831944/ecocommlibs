    public abstract class Recognizer
extends HandwritingContext
implements IRunnable, IRecognitionResultProvider
    The Recognizer class is the abstract base class of all the recognizers provided by the MyScript Handwriting Recognition technology.

    The Recognizer class implements the IRunnable interface. The run method is used to initiate the recognition process.

    Please note that the implementation of the Recognizer class is not synchronized. If multiple threads concurrently try to change the state of a Recognizer object, explicit synchronization must be done externally.
