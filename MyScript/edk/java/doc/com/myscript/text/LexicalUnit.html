    public final class LexicalUnit
extends Vocabulary
implements ICompilable
    The LexicalUnit class models the MyScript Handwriting Recognition Lexical Unit resource type.

    Lexical Units are used to constrain the handwriting recognition process by the means of regular expressions.

    A LexicalUnit object is serializable, in other words you can invoke the load and store methods on it.

    Please note that the implementation of the LexicalUnit class is not synchronized. If multiple threads concurrently try to change the state of a LexicalUnit object, explicit synchronization must be done externally.
