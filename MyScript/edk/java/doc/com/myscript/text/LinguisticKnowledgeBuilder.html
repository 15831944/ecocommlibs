Class LinguisticKnowledgeBuilder

  • java.lang.Object
    • com.myscript.engine.EngineObject
      • com.myscript.text.LinguisticKnowledgeBuilder
  • All Implemented Interfaces:
    IDisposable, IEngineObject, java.lang.Cloneable

    public final class LinguisticKnowledgeBuilder
extends EngineObject
    The LinguisticKnowledgeBuilder class models an object capable of building a Resource object containing linguistic knowledge derived from user input.

    Please note that the implementation of the LinguisticKnowledgeBuilder class is not synchronized. If multiple threads concurrently try to change the state of a LinguisticKnowledgeBuilder object, explicit synchronization must be done externally.
