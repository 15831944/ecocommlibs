    public final class IsolatedAlphabetKnowledge
extends AlphabetKnowledge
    The IsolatedAlphabetKnowledge class models the MyScript Handwriting Recognition Isolated Alphabet Knowledge resource type.

    Isolated Alphabet Knowledge resources embed knowledge about the isolated handwriting style according to a specified language / country.

    An IsolatedAlphabetKnowledge object is serializable, in other words you can invoke the load and store methods on it.

    Please note that the implementation of the IsolatedAlphabetKnowledge class is not synchronized. If multiple threads concurrently try to change the state of a IsolatedAlphabetKnowledge object, explicit synchronization must be done externally.
