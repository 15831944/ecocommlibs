Class CandidateIterator

  • java.lang.Object
    • com.myscript.engine.EngineObject
      • com.myscript.engine.Iterator
        • com.myscript.text.CandidateIterator
  • All Implemented Interfaces:
    IDisposable, IEngineObject, java.lang.Cloneable

    public final class CandidateIterator
extends Iterator
    The CandidateIterator class models an iterator over the candidates of a RecognitionResult.

    Please note that the implementation of the CandidateIterator class is not synchronized. If multiple threads concurrently try to change the state of a CandidateIterator object, explicit synchronization must be done externally.
