    public final class StructuredInputRecognizer
extends Recognizer
    The StructuredInputRecognizer class models a MyScript handwriting recognizer fueled by a structured input.

    Please note that the implementation of the StructuredInputRecognizer class is not synchronized. If multiple threads concurrently try to change the state of a StructuredInputRecognizer object, explicit synchronization must be done externally.
