    public final class Lexicon
extends Vocabulary
implements ICompilable
    The Lexicon class models the MyScript Handwriting Recognition Lexicon resource type.

    Lexicons are used to constrain the handwriting recognition process by allowing recognition of only the lexicon entries.

    A Lexicon object is serializable, in other words you can invoke the load and store methods on it.

    Please note that the implementation of the Lexicon class is not synchronized. If multiple threads concurrently try to change the state of a Lexicon object, explicit synchronization must be done externally.
