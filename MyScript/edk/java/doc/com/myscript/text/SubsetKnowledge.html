Class SubsetKnowledge

  • java.lang.Object
    • com.myscript.engine.EngineObject
      • com.myscript.text.Resource
        • com.myscript.text.SubsetKnowledge
  • All Implemented Interfaces:
    IDisposable, IEngineObject, java.lang.Cloneable

    public final class SubsetKnowledge
extends Resource
    The SubsetKnowledge class models the MyScript Handwriting Recognition Subset Knowledge resource type.

    Subset Knowledge resources are used to disable entries in the Alphabet Knowledge resources.

    Please note that the implementation of the SubsetKnowledge class is not synchronized. If multiple threads concurrently try to change the state of a SubsetKnowledge object, explicit synchronization must be done externally.
