Class OccurrenceIterator

  • java.lang.Object
    • com.myscript.engine.EngineObject
      • com.myscript.engine.Iterator
        • com.myscript.inksearch.OccurrenceIterator
  • All Implemented Interfaces:
    IDisposable, IEngineObject, java.lang.Cloneable

    public final class OccurrenceIterator
extends Iterator
    The OccurrenceIterator class models an iterator over the occurences of a FindResult object.

    Please note that the implementation of the OccurrenceIterator class is not synchronized. If multiple threads concurrently try to change the state of an OccurrenceIterator object, explicit synchronization must be done externally.
