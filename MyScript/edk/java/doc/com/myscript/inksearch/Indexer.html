    public abstract class Indexer
extends HandwritingContext
implements IRunnable, IIndexingResultProvider
    The Indexer class is the abstract base class for all the MyScript InkSearch indexing handwriting context entities.

    Indexers are responsible for the pre-processing phase that consists in analysing multiple digital ink samples and compiling the result of this analysis into an Index object, which will later be used to conduct the search process.

    The Indexer class implements the IRunnable interface: the run method is used to initiate the indexing process.

    The Indexer class implements the IIndexingResultProvider interface: the getResult method retrieve the indexing process result.

    Please note that the implementation of the Indexer class is not synchronized. If multiple threads concurrently try to change the state of an Indexer object, explicit synchronization must be done externally.
