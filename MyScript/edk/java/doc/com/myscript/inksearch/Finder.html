com.myscript.inksearch

Class Finder

  • java.lang.Object
    • com.myscript.engine.EngineObject
      • com.myscript.inksearch.Finder
  • All Implemented Interfaces:
    IAttachTarget, IDisposable, IEngineObject, IRunnable, IFindResultProvider, java.lang.Cloneable

    public final class Finder
extends EngineObject
implements IAttachTarget, IRunnable, IFindResultProvider
    The Finder class models the MyScript InkSearch Finder type that allows you to perform search requests on indexed digital ink samples.

    To perform an ink search request, you have to attach an Index object (mandatory) and a Query object (mandatory) and use the run method to initiate the search process.

    The Finder class implements the IAttachTarget interface: the Attach method is used to setup a Finder object with a Query object and an Index object.

    The Finder class implements the IRunnable interface: the run method is used to initiate the search process.

    The Finder class implements the IFindResultProvider interface: the getResult method is used to retrieve the result of the search process and returns a FindResult object.
