Package com.myscript.inksearch

The com.myscript.inksearch package provides classes and interfaces that model the access to the MyScript InkSearch technology.

See: Description

  • Interface Summary 
    Interface Description
    IFindResultProvider
    The IFindResultProvider interface models an EngineObject object that is able to provide the result of a search query, in other words on which you can invoke the getResult method that returns a FindResult object.
    IIndexingResultProvider
    The IIndexingResultProvider interface models an EngineObject object that is able to provide the result of an indexing process, in other words on which you can invoke the getResult method which returns an Index object.
  • Class Summary 
    Class Description
    AndNotQuery
    The AndNotQuery class models query that does a logical AND NOT operation between two Query objects.
    AndQuery
    The AndQuery class models query that does a logical AND operation between two Query objects.
    Finder
    The Finder class models the MyScript InkSearch Finder type that allows you to perform search requests on indexed digital ink samples.
    FindResult
    The FindResult class models the result of a search query.
    Index
    The Index class models the MyScript InkSearch index type.
    Indexer
    The Indexer class is the abstract base class for all the MyScript InkSearch indexing handwriting context entities.
    NearQuery
    The NearQuery class models query that does a logical AND operation between two Query objects and also introduces a notion of proximity.
    OccurrenceCharacterIterator
    The OccurrenceCharacterIterator class models an iterator over the characters of the label of a search occurrence.
    OccurrenceIterator
    The OccurrenceIterator class models an iterator over the occurences of a FindResult object.
    OrQuery
    The OrQuery class models query that does a logical OR operation between two Query objects.
    Query
    The Query class is the abstract base class for all the kind of MyScript InkSearch queries.
    SizeAccuracyCompromise
    The SizeAccuracyCompromise class is a type safe enumeration type that specifies the different compromise level used to control the balance between index size and search accuracy.
    StringQuery
    The StringQuery class models a query that allows the user to search all the occurrences that match a specified character string.
    StructuredInputIndexer
    The StructuredInputIndexer class models an handwriting indexer fueled by a structured input.
    StructuredInputSlimIndexer
    The StructuredInputSlimIndexer class models a slim handwriting indexer fueled by a structured input.
    TextIndexer
    The TextIndexer class models an handwriting indexer that processes MyScript document content fields instead of the legacy input.
    VocabularyQuery
    The VocabularyQuery class models a query that allows the user to search all the occurrences that matches the content of a specified Vocabulary resource object.
  • Exception Summary 
    Exception Description
    MissingIndexException
    The MissingIndexException exception class models a situation where the MyScript Engine failed to perform the requested operation because an Index object is missing.
    MissingQueryException
    The MissingQueryException exception class models a situation where the MyScript Engine failed to perform the requested operation because a Query object is missing.

Package com.myscript.inksearch Description

The com.myscript.inksearch package provides classes and interfaces that model the access to the MyScript InkSearch technology.

The com.myscript.inksearch package depends on the com.myscript.engine package for interaction with the native side of MyScript Engine and on the com.myscript.text package that allows access to the MyScript Handwriting Recognition technology.
