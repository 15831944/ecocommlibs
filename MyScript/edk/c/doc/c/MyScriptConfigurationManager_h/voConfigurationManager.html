voConfigurationManager reference type.

The configuration manager provides easy configuration of all MyScript processing units from Text to Shape, Math, Music, and Analyzer. It manages a set of configuration bundles defined through search directories to leverage named configurations. A set of standard configurations is provided as part of the MyScript SDK in the conf/text, conf/shape, ... directories. You can modify and/or add configurations to adjust them to your specific needs.

The actual configuration is defined through a simple scripting language similar to the scripts in InkTool. Below is a list of the supported commands:

When executing a configuration script a configurer will replace ${[_A-Za-z][_0-9A-Za-z]*} patterns with the value of the environment variable with the name between the curly braces.

By default, the environment contains a single variable named "lite", which value is "" on desktop platforms and ".lite" on mobile platforms. We set this so that provided configuration bundles load the *.lite.res resources on mobiles and the *.res resources on desktops, by default.

You can override this behavior by setting the "lite" variable to the other value. Of course, this function will allow you to set the "lite" variable with any other value but applying standard configurations will then stop working.
