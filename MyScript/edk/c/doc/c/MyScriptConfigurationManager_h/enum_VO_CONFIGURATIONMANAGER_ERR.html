Enumeration VO_CONFIGURATIONMANAGER_ERR

#include <c/MyScriptConfigurationManager.h>
enum _VO_CONFIGURATIONMANAGER_ERR

Lists the MyScript Configuration Manager errors.

Value Summary
VO_CONFIGURATION_BUNDLE_NOT_FOUND= VO_INVALID_STATE | (8 << 16)
          The configuration bundle name does not correspond to a .conf file in the configuration manager search directories.
VO_CONFIGURATION_NOT_FOUND
          The configuration name does not correspond to an existing configuration in a bundle.
VO_RESOURCE_NOT_FOUND
          The configuration refers to a resource that cannot be found.
VO_INVALID_CONFIGURATION
          The configuration contains incorrect commands.
