Structure _voILayerIterator
#include <c/MyScriptDocument.h>
struct _voILayerIterator
 extends voIIterator

Functions composing the voILayerIterator interface.

This interface exposes the public methods of the voLayerIterator type.

Note:

Layer iterators created from a voPageSelection are not allowed to modify the layout.

As the C language does not provide any mechanism that could be used to simulate inheritance, inherited members are simply copied from the base structure.
