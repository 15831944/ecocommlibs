setPrimitiveData

bool setPrimitiveData(voEngine engine, voTypesetItem target, void *data, size_t sizeOfData)
Modifies the geometry of the primitive associated with this typeset item.

Parameters:
engine

the engine.

target

the target typeset item.

data

the new characteristics of the primitive, described as a voCenterEllipticArc or a voEndpointEllipticArc for a voArcPrimitive or as a voLineSegment for a voLinePrimitive.

sizeOfData

the size of data in bytes.
Returns:

true on success, otherwise false.

Throws:
VO_NO_SUCH_ENGINE

when the engine reference is invalid.

VO_NO_SUCH_OBJECT

when the target reference is invalid.

VO_INVALID_OPERATION

when target is not a voTypesetItem of type VO_LinePrimitive or VO_ArcPrimitive.

VO_INVALID_USER_BUFFER

when data is not readable.

VO_INVALID_ARGUMENT

when data contains invalid values.

VO_INVALID_ARGUMENT

when sizeOfData does not match the size of a voCenterEllipticArc, a voEndpointEllipticArc or a voLineSegment.
