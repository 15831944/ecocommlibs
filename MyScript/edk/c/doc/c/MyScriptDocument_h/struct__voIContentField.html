setConfiguration

bool setConfiguration(voEngine engine, voContentField target, voCharset charset, const voString *bundleName, const voString *configurationName, const voString *configurationScript)
Configures the configurations bundle, the configuration name and the configuration script of this content field.

These configuration strings may be set to NULL, as long as the following rules are followed:

  • If bundleName is NULL, then configurationName must be NULL and configurationScript must not be NULL.

  • If configurationName is NULL, then bundleName must be NULL and configurationScript must not be NULL.

  • If configurationScript is NULL, then bundleName and configurationName must not be NULL.

Parameters:
engine

the engine.

target

the target content field.

charset

the charset used to encode parameter strings.

bundleName

the name of the configurations bundle, or NULL.

configurationName

the name of the configuration, or NULL.

configurationScript

the configuration script, or NULL.
Returns:

true on success, otherwise false.

Throws:
VO_NO_SUCH_ENGINE

when the engine reference is invalid.

VO_NO_SUCH_OBJECT

when the target reference is invalid.

VO_INVALID_OPERATION

when target is not a voContentField.

VO_NO_SUCH_OBJECT

when the charset reference is invalid.

VO_NO_SUCH_OBJECT

when the charset reference is NULL and no default charset was set.

VO_INVALID_OBJECT

when charset is not a voCharset.

VO_INVALID_USER_BUFFER

when bundleName or its bytes member is not readable.

VO_INVALID_USER_BUFFER

when configurationName or its bytes member is not readable.

VO_INVALID_USER_BUFFER

when configurationScript or its bytes member is not readable.

VO_INVALID_ARGUMENT

when the combination of bundleName, configurationName and configurationScript is not allowed.
Since:

7.0_0
