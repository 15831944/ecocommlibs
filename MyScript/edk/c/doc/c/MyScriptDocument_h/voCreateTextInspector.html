c/MyScriptDocument.h
Function voCreateTextInspector

#include <c/MyScriptDocument.h>
VO_INLINE voTextInspector voCreateTextInspector(voEngine engine, voPage page)

Creates a new voTextInspector instance.

Parameters:
engine

the engine.

page

the associated page.
Returns:

the newly created voTextInspector instance on success, otherwise NULL.

Throws:
VO_NO_SUCH_ENGINE

when the engine reference is invalid.

VO_NO_SUCH_OBJECT

when the page reference is invalid.

VO_INVALID_OBJECT

when page is not a voPage.

VO_LIMIT_EXCEEDED

when the number of voTextInspector instances would exceed the allowed limit (please remember that the limit can be shared by several types).
