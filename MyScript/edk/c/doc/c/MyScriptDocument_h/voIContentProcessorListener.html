Type voIContentProcessorListener

voIContentProcessorListener
#include <c/MyScriptDocument.h>
typedef struct _voIContentProcessorListener voIContentProcessorListener

Functions composing the voIContentProcessorListener interface.

Note:

This interface is implementable by user objects. User code is not allowed to invoke this interface directly: as a consequence voGetInterface() returns NULL.

Since:

7.0_0

void onResultAvailable(voEngine engine, voUserObject target, voContentField field, voPageChangeSet changeSet)
