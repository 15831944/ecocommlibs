#include <c/MyScriptDocument.h>
typedef struct _voIPageUpdateListener voIPageUpdateListener

Functions composing the voIPageUpdateListener interface.

Note:

This interface is implementable by user objects. User code is not allowed to invoke this interface directly: as a consequence voGetInterface() returns NULL.
