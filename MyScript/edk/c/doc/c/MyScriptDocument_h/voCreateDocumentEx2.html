c/MyScriptDocument.h
Function voCreateDocumentEx2

#include <c/MyScriptDocument.h>
VO_INLINE voDocument voCreateDocumentEx2(voEngine engine, voCharset charset, const voString *fileName, const voString *temporaryDirectoryName, bool createDirectory, bool overwrite)

Creates a new voDocument instance.

Parameters:
engine

the engine.

charset

the charset used to encode parameter strings.

fileName

the pathname of the document.

temporaryDirectoryName

the pathname of the temporary directory to use. If empty, a directory name will be set to fileName + "-files".

createDirectory

true when parent directory of fileName must be created if it does not exist, otherwise false.

overwrite

true when a new document must be created even if fileName points to an existing document, otherwise false.
Returns:

the newly created voDocument instance on success, otherwise NULL.

Throws:
VO_NO_SUCH_ENGINE

when the engine reference is invalid.

VO_NO_SUCH_OBJECT

when the charset reference is invalid.

VO_NO_SUCH_OBJECT

when the charset reference is NULL and no default charset was set.

VO_INVALID_OBJECT

when charset is not a voCharset.

VO_INVALID_USER_BUFFER

when fileName or its bytes member is not readable.

VO_INVALID_USER_BUFFER

when temporaryDirectoryName or its bytes member is not readable.

VO_IO_FAILURE

when fileName cannot be created or read.

VO_LIMIT_EXCEEDED

when the number of voDocument instances would exceed the allowed limit (please remember that the limit can be shared by several types).
