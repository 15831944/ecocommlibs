c/MyScriptDocument.h
Function voCreateDocumentFinder

#include <c/MyScriptDocument.h>
VO_INLINE voDocumentFinder voCreateDocumentFinder(voEngine engine, voDocument document, voDocumentFinderCallback callback, void *userParam)

Creates a new voDocumentFinder instance.

Parameters:
engine

the engine.

document

the document to search in.

callback

the callback that will receive worker thread events.

userParam

an optional pointer to a user defined parameter that will be passed back to the callback function.
Returns:

the newly created voDocumentFinder instance on success, otherwise NULL.

Throws:
VO_NO_SUCH_ENGINE

when the engine reference is invalid.

VO_NO_SUCH_OBJECT

when the document reference is invalid.

VO_INVALID_OBJECT

when document is not a voDocument.

VO_LIMIT_EXCEEDED

when the number of voDocumentFinder instances would exceed the allowed limit (please remember that the limit can be shared by several types).
