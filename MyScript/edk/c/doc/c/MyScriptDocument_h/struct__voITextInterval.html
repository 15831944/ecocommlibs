getBeginPosition

int32_t getBeginPosition(voEngine engine, voTextInterval target)
Returns the begin position of this text interval.

This position corresponds to the index of a recognized character in the content field associated with target.

Parameters:
engine

the engine.

target

the target text interval.
Returns:

the position on success, otherwise -1.

Throws:

VO_NO_SUCH_OBJECT

VO_INVALID_OPERATION

bool setBeginPosition(voEngine engine, voTextInterval target, int32_t position)
