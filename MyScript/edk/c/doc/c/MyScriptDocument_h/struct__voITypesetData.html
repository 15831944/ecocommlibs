apply

bool apply(voEngine engine, voTypesetData target, bool keepLayout, voPageSelection oldData, voPageSelection newData)
Applies the typesetting modifications described in the given typeset data.

If items that already reference a typeset result have been modified, a new glyph or primitive will be created, and the old one will be added to oldData.

Parameters:
engine

the engine.

target

the typeset data.

keepLayout

true when the layout information associated with the strokes to be typeset should be transferred to their typeset counterpart.

oldData

a selection will be filled with the strokes that have been replaced by typeset data, or NULL.

newData

a selection will be filled with the primitives or glyphs generated by the typeset operation, or NULL.
Returns:

true on success, otherwise false.

Throws:
VO_NO_SUCH_ENGINE

when the engine reference is invalid.

VO_NO_SUCH_OBJECT

when the target reference is invalid.

VO_INVALID_OPERATION

when target is not a voTypesetData.

VO_NOT_PERMITTED_FROM_THIS_THREAD

when the apply() call was made from within a voContentProcessor callback.

VO_INVALID_STATE

when the content field associated with target has been modified since target was created (e.g. by applying a voPageChangeSet or by processing that content field).
