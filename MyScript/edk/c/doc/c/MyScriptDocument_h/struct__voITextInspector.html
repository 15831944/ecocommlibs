expand

bool expand(voEngine engine, voTextInspector target, voTextInterval interval, voTextBoundary beginBoundary, voTextBoundary endBoundary)
Expands the specified text interval using the specified limits.

Parameters:
engine

the engine.

target

the target text inspector.

interval

the text interval to be expanded.

beginBoundary

the boundary to which the begin position should be expanded.

endBoundary

the boundary to which the end position should be expanded.
Returns:

true on success, otherwise false.

Throws:
VO_NO_SUCH_ENGINE

when the engine reference is invalid.

VO_NO_SUCH_OBJECT

when the target reference is invalid.

VO_INVALID_OPERATION

when target is not a voTextInspector.

VO_NO_SUCH_OBJECT

when the interval reference is invalid.

VO_INVALID_OPERATION

when interval is not a voTextInterval.

VO_INDEX_OUT_OF_BOUNDS

when beginBoundary or endBoundary is out of bounds.

VO_INVALID_ARGUMENT

when interval refers to a content field that has been removed.

fromField

voTextInterval fromField(voEngine engine, voTextInspector target, voContentField field)
Returns a text interval that corresponds to the specified text content field.

The interval surrounds the whole field: the begin position is zero and the end position is the number of characters in the field.

Parameters:
engine

the engine.

target

the target text inspector.

field

a text content field.
Returns:

the newly created voTextInterval instance on success, otherwise NULL.

Throws:
VO_NO_SUCH_ENGINE

when the engine reference is invalid.

VO_NO_SUCH_OBJECT

when the target reference is invalid.

VO_INVALID_OPERATION

when target is not a voTextInspector.

VO_NO_SUCH_OBJECT

when the field reference is invalid.

VO_INVALID_OBJECT

when field is not a voContentField of type VO_CONTENT_FIELD_TEXT.

VO_INVALID_ARGUMENT

when field comes from a different page.

VO_INVALID_ARGUMENT

when field refers to a content field that has been removed.

VO_LIMIT_EXCEEDED

when the number of voTextInterval instances would exceed the allowed limit (please remember that the limit can be shared by several types).

hitChar

voTextInterval hitChar(voEngine engine, voTextInspector target, const voPoint *position, voContentField field)
Returns a text interval that corresponds to the character under the specified position.

The interval surrounds the character so the begin position points to it and the end position points just after it.

Note:

A text content field can be specified to constrain the search.

Parameters:
engine

the engine.

target

the target text inspector.

position

the geometric position.

field

a text content field, or NULL.
Returns:

the newly created voTextInterval instance, if found. When NULL, use voGetError() to figure out whether an error occurred.

Throws:
VO_NO_SUCH_ENGINE

when the engine reference is invalid.

VO_NO_SUCH_OBJECT

when the target reference is invalid.

hitCursor

voTextInterval hitCursor(voEngine engine, voTextInspector target, const voPoint *position, voContentField field)
Returns a text interval that corresponds to the closest character boundary from the specified position.

The interval is a cursor position so the end position will be the same as the begin position.

Note:

A text content field can be specified to constrain the search. Won't hit when position is too far from the field boundaries.

Parameters:
engine

the engine.

target

the target text inspector.

position

the geometric position.

field

a text content field, or NULL.
Returns:

the newly created voTextInterval instance, if found. When NULL, use voGetError() to figure out whether an error occurred.

Throws:
VO_NO_SUCH_ENGINE

when the engine reference is invalid.

VO_NO_SUCH_OBJECT

when the target reference is invalid.

VO_INVALID_OPERATION

when target is not a voTextInspector.

VO_INVALID_USER_BUFFER

when position is not readable.

VO_NO_SUCH_OBJECT

VO_INVALID_OBJECT

when field is not a voContentField of type VO_CONTENT_FIELD_TEXT.

VO_INVALID_ARGUMENT

when field comes from a different page.

VO_INVALID_ARGUMENT
