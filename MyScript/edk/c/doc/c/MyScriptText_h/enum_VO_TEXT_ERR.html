Value Detail

VO_MISSING_ALPHABET_KNOWLEDGE

VO_MISSING_ALPHABET_KNOWLEDGE
Unable to complete the requested operation because a voAlphabetKnowledge resource is missing.

This does not necessarily mean that no voAlphabetKnowledge resource is attached. For example, if you fill a voStructuredInput with VO_MULTI_LINE_TEXT input units and you attach only a voIsolatedAlphabetKnowledge you will get this error.

See also:
voAlphabetKnowledge

VO_MISSING_SOURCE

VO_MISSING_SOURCE
Unable to complete the requested operation because the input consumer source is missing.

See also:
voIInputConsumer

VO_CHAR_STROKE_LIMIT_EXCEEDED

VO_CHAR_STROKE_LIMIT_EXCEEDED
One of the character input units within an input exceeds the stroke limit fixed by the recognizer.

Note:

This limit may change from one recognizer configuration to another.

VO_REGEX_SUBEXP_IMBALANCE

VO_REGEX_SUBEXP_IMBALANCE
Parse error in a regular expression: sub-expression imbalance.

This means that the number of open and close parentheses does not match.

See also:
voLexicalUnit

VO_REGEX_UNMATCHED_OPEN_SET

VO_REGEX_UNMATCHED_OPEN_SET
Parse error in a regular expression: unmatched open set.

This means that an open square bracket exists without the matching close square bracket.

See also:
voLexicalUnit

VO_REGEX_UNMATCHED_OPEN_REF

VO_REGEX_UNMATCHED_OPEN_REF
Parse error in a regular expression: unmatched open reference.

This means that an open brace exists without the matching close brace.

See also:
voLexicalUnit

VO_REGEX_TRAILING_ESCAPE

VO_REGEX_TRAILING_ESCAPE
Parse error in a regular expression: trailing escape.

This means that a back slash was found at the end of the expression (without the escaped character).

See also:
voLexicalUnit

VO_REGEX_INVALID_CHARACTER

VO_REGEX_INVALID_CHARACTER
Parse error in a regular expression: invalid character.

This means that an invalid character was found.

See also:
voLexicalUnit

VO_REGEX_INVALID_REFERENCE

VO_REGEX_INVALID_REFERENCE
Parse error in a regular expression: invalid reference.

This means that a reference was found but the corresponding definition does not exist.

See also:
voLexicalUnit

VO_REGEX_UNMATCHED_CLOSE_SET

VO_REGEX_UNMATCHED_CLOSE_SET

VO_REGEX_UNESCAPED_OPERATOR

VO_REGEX_UNESCAPED_OPERATOR
Parse error in a regular expression: unescaped operator.

This means that a regular expression operator was found when a normal character was expected.

Escape it using the backslash character if you meant it literally.

See also:
voLexicalUnit
Since:
