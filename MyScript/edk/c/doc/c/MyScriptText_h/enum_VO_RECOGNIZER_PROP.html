VO_CHAR_CANDIDATE_LIST_SIZE

VO_CHAR_CANDIDATE_LIST_SIZE
Specifies the size of the candidate lists that will be provided at the character level in the recognition result.

This value must be between 1 and 20.

Usage: 
uint32_t value;
voGetProperty(engine, object, VO_CHAR_CANDIDATE_LIST_SIZE, &value, sizeof(value));
voSetProperty(engine, object, VO_CHAR_CANDIDATE_LIST_SIZE, &value, sizeof(value));
