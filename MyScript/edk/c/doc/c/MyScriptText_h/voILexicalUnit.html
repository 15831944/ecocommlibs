defineMainRegex

bool defineMainRegex(voEngine engine, voLexicalUnit target, voCharset charset, const voString *regex, float frequency)
Defines the main regular expression.

Parameters:
engine

the engine.

target

the target lexical unit.

charset

the charset used to encode parameter strings.

regex

the regular expression.

frequency

the frequency assigned to this regular expression. This value must be between -1 (low frequency) and 1 (high frequency).
Returns:

true on success, otherwise false.

Throws:
VO_NO_SUCH_ENGINE

when the engine reference is invalid.

VO_NO_SUCH_OBJECT

when the target reference is invalid.

VO_INVALID_OPERATION

when target is not a voLexicalUnit.

VO_MODIFICATION_ACCESS_DENIED

when target is in use by another object.

VO_NO_SUCH_OBJECT

when the charset reference is invalid.

VO_NO_SUCH_OBJECT

when the charset reference is NULL and no default charset was set.

VO_INVALID_OBJECT

when charset is not a voCharset.

VO_INVALID_USER_BUFFER

when regex or its bytes member is not readable.

VO_INVALID_ARGUMENT

when regex is empty.

VO_REGEX_INVALID_CHARACTER

when name contains invalid characters.

VO_REGEX_SUBEXP_IMBALANCE

when regex contains a sub-expression imbalance.

VO_REGEX_UNMATCHED_OPEN_SET

when regex contains an unmatched open set.

VO_REGEX_UNMATCHED_OPEN_REF

when regex contains an unmatched open reference.

VO_REGEX_TRAILING_ESCAPE

when regex contains a trailing escape character.

VO_REGEX_INVALID_CHARACTER

when regex contains an invalid character.

VO_REGEX_INVALID_REFERENCE

when regex refers to an undefined entity.

VO_REGEX_UNMATCHED_CLOSE_SET

when regex contains an unmatched close set

VO_REGEX_UNMATCHED_CLOSE_REF

when regex contains an unmatched close reference.

VO_REGEX_UNESCAPED_OPERATOR

when regex contains an unescaped operator.

VO_INVALID_ARGUMENT

when frequency is not between -1 and 1.

VO_INVALID_STATE

when target is compiled.
Note:

This is the regular expression that defines the lexical unit itself. This means that you will get a VO_INVALID_STATE error if you try to compile a voLexicalUnit with only named regular expressions defined.
