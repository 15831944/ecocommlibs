Value Detail

VO_USER_HWCTX_LIMIT

VO_USER_HWCTX_LIMIT
Represents the maximum number of voHandwritingContext instances that can exist at the same time in the user side of a MyScript enabled application.

The special value zero means no limit. Other values are adjusted to the next multiple of 16. A value under the current count is permitted and will constrain future creations.

Usage: 
uint32_t value;
voGetTypeProperty(engine, VO_HandwritingContext, VO_USER_HWCTX_LIMIT, &value, sizeof(value));
voSetTypeProperty(engine, VO_HandwritingContext, VO_USER_HWCTX_LIMIT, &value, sizeof(value));
