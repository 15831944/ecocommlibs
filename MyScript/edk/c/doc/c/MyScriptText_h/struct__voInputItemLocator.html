Structure _voInputItemLocator
#include <c/MyScriptText.h>
struct _voInputItemLocator

Represents the location of an item in the input.

An input item is either a point within a stroke component, or a character within a string component.

Then each component is located within an input unit.

Member Summary
shortcomponentIndex
           Index of the stroke component or the string component that contains the item, within its unit.
shortitemIndex
           Index of the point or character item, within its component.
shortunitIndex
           Index of the input unit that contains the item.
