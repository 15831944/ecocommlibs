matches

voYesNo matches(voEngine engine, voDigitalTextMatcher target, voCharset charset, const voString *string)
Answers the question: does the given string match the current resource configuration?

Parameters:
engine

the engine.

target

the target digital text matcher.

charset

the charset used to encode parameter strings.

string

the string to be tested.
Returns:

VO_YES or VO_NO on success, otherwise VO_ERR.

Throws:
VO_NO_SUCH_ENGINE

when the engine reference is invalid.

VO_NO_SUCH_OBJECT

when the target reference is invalid.

VO_INVALID_OPERATION

when target is not a voDigitalTextMatcher.

VO_NO_SUCH_OBJECT

when the charset reference is invalid.

VO_NO_SUCH_OBJECT

when the charset reference is NULL and no default charset was set.

VO_INVALID_OBJECT

when charset is not a voCharset.

VO_INVALID_USER_BUFFER

when string or its bytes member is not readable.

VO_INVALID_ARGUMENT

when string is empty.
