voYesNo(*equals)(voEngine engine, voIterator target, voIterator other)
           Tells whether the two iterators are equal.

Two iterators are considered equal if they are positioned at the same position and iterate on the same collection. This function will also return VO_YES if both iterators are NULL or if one of the iterators is NULL and the other is positioned at the end of the iteration.
