bool addWord(voEngine engine, voLexicon target, voCharset charset, const voString *word)
Adds a word to this lexicon.

Parameters:
engine

the engine.

target

the target lexicon.

charset

the charset used to encode parameter strings.

word

the word to be added.
Returns:

true on success, otherwise false.

Throws:
VO_NO_SUCH_ENGINE

when the engine reference is invalid.

VO_NO_SUCH_OBJECT

when the target reference is invalid.

VO_INVALID_OPERATION

when target is not a voLexicon.

VO_NO_SUCH_OBJECT

when the charset reference is invalid.

VO_NO_SUCH_OBJECT

when the charset reference is NULL and no default charset was set.

VO_INVALID_OBJECT

when charset is not a voCharset.

VO_INVALID_USER_BUFFER

when word or its bytes member is not readable.

VO_INVALID_ARGUMENT

when word is the empty string or only contain space characters.

VO_INVALID_STATE

when target is compiled.
