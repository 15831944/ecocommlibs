getGlobalScore

float getGlobalScore(voEngine engine, voFindResult target)
Returns the global search result score, which is a combination of all the occurrence scores.

The score varies from 0.0f (low probability) to 1.0f (high probability).

Parameters:
engine

the engine.

target

the target find result.
Returns:

the global search score on success, otherwise -1.

Throws:
VO_NO_SUCH_ENGINE

when the engine reference is invalid.

VO_NO_SUCH_OBJECT

when the target reference is invalid.

VO_INVALID_OPERATION

when target is not a voFindResult.
