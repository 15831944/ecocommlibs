c/MyScriptInkSearch.h
Function voCreateVocabularyQuery

#include <c/MyScriptInkSearch.h>
VO_INLINE voVocabularyQuery voCreateVocabularyQuery(voEngine engine, voVocabulary vocabulary)

Creates a new voVocabularyQuery instance.

Parameters:
engine

the engine.

vocabulary

the associated vocabulary.
Returns:

the newly created voVocabularyQuery instance on success, otherwise NULL.

Throws:
VO_NO_SUCH_ENGINE

when the engine reference is invalid.

VO_NO_SUCH_OBJECT

when the vocabulary reference is invalid.

VO_INVALID_OBJECT

when vocabulary is not a voVocabulary.

VO_NOT_COMPILED

when vocabulary is not compiled.

VO_LIMIT_EXCEEDED

when the number of voVocabularyQuery instances would exceed the allowed limit (please remember that the limit can be shared by several types).
