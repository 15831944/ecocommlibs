c/MyScriptInkSearch.h
Function voCreateBinaryQuery

#include <c/MyScriptInkSearch.h>
VO_INLINE voQuery voCreateBinaryQuery(voEngine engine, voTypeId type, voQuery query1, voQuery query2)

Creates a new binary query of the specified type.

Parameters:
engine

the engine.

type

the type of query to be created: VO_OrQuery, VO_AndQuery, VO_AndNotQuery or VO_NearQuery.

query1

the first query.

query2

the second query.
Returns:

the newly created voQuery instance on success, otherwise NULL.

Throws:
VO_NO_SUCH_ENGINE

when the engine reference is invalid.

VO_NO_SUCH_TYPE

when the specified type is invalid.

VO_TYPE_NOT_CONCRETE

when the specified type is not concrete.

VO_NO_SUCH_CONSTRUCTOR

when the specified type does not have a constructor with the given initializer.

VO_NO_SUCH_OBJECT

when the query1 reference is invalid.

VO_INVALID_OBJECT

when query1 is not a voQuery.

VO_NO_SUCH_OBJECT

when the query2 reference is invalid.

VO_INVALID_OBJECT

when query2 is not a voQuery.

VO_LIMIT_EXCEEDED

when the number of voQuery instances would exceed the allowed limit (please remember that the limit can be shared by several types).
