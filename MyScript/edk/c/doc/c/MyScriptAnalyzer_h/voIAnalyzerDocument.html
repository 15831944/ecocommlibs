addStroke

bool addStroke(voEngine engine, voAnalyzerDocument target, const float *x, size_t xByteStride, const float *y, size_t yByteStride, int32_t pointCount)
Adds a pen stroke to this document.

Parameters:
engine

the engine.

target

the target document.

x

a pointer to the first x value.

xByteStride

the stride in bytes between each x value.

y

a pointer to the first y value.

yByteStride

the stride in bytes between each y value.

pointCount

the number of points.
Returns:

true on success, otherwise false.

Throws:
VO_NO_SUCH_ENGINE

when the engine reference is invalid.

VO_NO_SUCH_OBJECT

when the target reference is invalid.

VO_INVALID_OPERATION

when target is not a voAnalyzerDocument.

VO_INVALID_ARGUMENT

when xByteStride == 0.

VO_INVALID_ARGUMENT

when yByteStride == 0.

VO_INVALID_ARGUMENT

when pointCount <= 0.

VO_INVALID_USER_BUFFER

when the combination of x, xByteStride, y, yByteStride and pointCount leads to invalid memory access.

VO_LIMIT_EXCEEDED

when pointCount > 32767.

getStrokeAt

bool getStrokeAt(voEngine engine, voAnalyzerDocument target, int32_t index, float *x, size_t xByteStride, float *y, size_t yByteStride, int32_t *pointCount)
Returns the points of the stroke at the specified index.

Parameters:
engine

the engine.

target

the target document.

index

the index of the requested stroke.

x

the recipient for the x values.

xByteStride

the stride in bytes between each x value.

y

the recipient for the y values.

yByteStride

the stride in bytes between each y value.

pointCount

on input, the capacity of the x and y recipients, on output, the recipient for the number of points in the stroke.
Returns:

true on success, otherwise false.

Throws:
VO_NO_SUCH_ENGINE

when the engine reference is invalid.

VO_NO_SUCH_OBJECT

when the target reference is invalid.

VO_INVALID_OPERATION

when target is not a voAnalyzerDocument.

VO_INDEX_OUT_OF_BOUNDS

when index is out of bounds.

VO_INVALID_USER_BUFFER

when pointCount is not readable or writable.

VO_INVALID_ARGUMENT

when x or y is not NULL and pointCount <= 0

VO_INVALID_USER_BUFFER

when the combination of x, xByteStride, y, yByteStride and pointCount leads to invalid memory access.
Note:

You can pass NULL in place of x and y to retrieve the number of points in the stroke.

setCoordinateResolution

bool setCoordinateResolution(voEngine engine, voAnalyzerDocument target, float resolution)
Sets the resolution of the coordinates in dpi.

This is the resolution of the grid of integer values in the chosen coordinate system. For example, if you use floating-point pixel coordinates you must give the screen resolution, no matter the resolution of the capturing device.

Parameters:
engine

the engine.

target

the target document.

resolution

the coordinate resolution.
Returns:

true on success, otherwise false.

Throws:
VO_NO_SUCH_ENGINE

when the engine reference is invalid.

VO_NO_SUCH_OBJECT

when the target reference is invalid.

VO_INVALID_OPERATION

when target is not a voAnalyzerDocument.

VO_INVALID_ARGUMENT

when resolution is not strictly positive.
