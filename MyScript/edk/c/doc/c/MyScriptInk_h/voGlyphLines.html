Member Detail

baseline

float baseline
The ratio between the distance from the top side of the glyph to the baseline, and the height of the glyph: baseline = (baseline_position - top_of_glyph) / glyph_height.

This ratio can be greater than 1 or negative if this glyph is not properly placed on a long text line.

x_height

float x_height
The positive ratio between the height of the letter 'x', and the height of the glyph : x_height = height_of_the_letter_x / glyph_height.

Note:

0 means unspecified.
