addStroke

bool addStroke(voEngine engine, voInk target, voInkStroke stroke)
Adds a stroke to this ink.

Note:

A call to this function may change the stroke format and values of the stroke being added: when the stroke format already in use in the ink differs from the stroke format of the stroke to be added, samples are converted to ensure that the units associated with each stroke format channel are the same for all strokes.

Parameters:
engine

the engine.

target

the target ink.

stroke

the stroke to be added.
Returns:

true on success, otherwise false.

Throws:
VO_NO_SUCH_ENGINE

when target is not a voInk.

VO_NO_SUCH_OBJECT

VO_INVALID_OBJECT

when stroke is not a voInkStroke.

VO_INVALID_ARGUMENT

getStrokeFromId

voInkStroke getStrokeFromId(voEngine engine, voInk target, int64_t id)
Returns the stroke associated with the specified identifier.

A stroke's identifier can be retrieved by calling getStrokeId().

When a stroke is chopped or erased (even partially), its identifier is invalidated.

Calling this function with an outdated identifier returns NULL.

Parameters:
engine

the engine.

target

the target ink.

id

the identifier of the stroke to be retrieved.
Returns:

the stroke associated with id if any, otherwise NULL. When NULL, use voGetError() to figure out whether an error occurred.

Throws:
VO_NO_SUCH_ENGINE

when the engine reference is invalid.

VO_NO_SUCH_OBJECT

when the target reference is invalid.

VO_INVALID_OPERATION

when target is not a voInk.

VO_INVALID_ARGUMENT

when id does not correspond to a possible stroke (e.g. it references a deleted stroke).
