rotate

bool rotate(voEngine engine, voTransform *transform, float a, float x0, float y0)
Multiplies the transform with a rotation transformation.

This is equivalent to calling multiply(R), where R is a voTransform represented by the following matrix:

  [   cos(a)   -sin(a)    -cos(a) * x0 + sin(a) * y0 + x0   ]
  [   sin(a)    cos(a)    -sin(a) * x0 - cos(a) * y0 + y0   ]
  [   0           0                      1                  ]

Parameters:
engine

the engine.

transform

the target transform.

a

the rotation angle in radians.

x0

the x position of the origin point.

y0

the y position of the origin point.
Returns:

true on success, otherwise false.

Throws:
VO_NO_SUCH_ENGINE

when the engine reference is invalid.

VO_INVALID_USER_BUFFER

when transform is not writable.
