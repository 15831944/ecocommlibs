addChannel

bool voIInkStrokeFormatBuilder::addChannel(voEngine engine, voInkStrokeFormatBuilder target, voCharset charset, const voString *name)
Adds a new channel to the stroke format to build.

Here below are the predefined names and their recognized meaning:

  Name | Description
  ---- | -----------
  X    | Horizontal position, increasing to the right.
  Y    | Vertical position, increasing downward.
  T    | Sample timestamp.
  F    | Tip force.
  W    | Stroke width.
  OTx  | Tilt along the X axis.
  OTy  | Tilt along the Y axis.
  OA   | Azimuth angle (yaw).
  OE   | Elevation angle (pitch).
  OR   | Rotation (counter-clockwise rotation around the pen axis).

You can also store application specific values by using a name not found in the table above.

Note:

A stroke format must at least contain X and Y channels.

Parameters:
engine

the engine.

target

the target stroke format builder.

charset

the charset used to encode parameter strings.

name

the name of channel to be added.
Returns:

true on success, otherwise false.

Throws:
VO_NO_SUCH_ENGINE

when the engine reference is invalid.

VO_NO_SUCH_OBJECT

when the target reference is invalid.

VO_INVALID_OPERATION

when target is not a voInkStrokeFormatBuilder.

VO_NO_SUCH_OBJECT

when the charset reference is invalid.

VO_NO_SUCH_OBJECT

when the charset reference is NULL and no default charset was set.

VO_INVALID_OBJECT

when charset is not a voCharset.

VO_INVALID_USER_BUFFER

when name or its bytes member is not readable.

VO_INVALID_STATE

when a channel with this name already exists.
