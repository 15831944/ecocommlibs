c/MyScriptInk.h
Function voCreateInkStrokeBuilder2

#include <c/MyScriptInk.h>
VO_INLINE voInkStrokeBuilder voCreateInkStrokeBuilder2(voEngine engine, voCharset charset, const voString *strokeFormatString)

Creates a new voInkStrokeBuilder instance.

Parameters:
engine

the engine.

charset

the charset used to encode parameter strings.

strokeFormatString

the format of the strokes built by the stroke builder.
Returns:

the newly created voInkStrokeBuilder instance on success, otherwise NULL.

Throws:
VO_NO_SUCH_ENGINE

when the engine reference is invalid.

VO_NO_SUCH_OBJECT

when the charset reference is invalid.

VO_NO_SUCH_OBJECT

when the charset reference is NULL and no default charset was set.

VO_INVALID_OBJECT

when charset is not a voCharset.

VO_INVALID_USER_BUFFER

when strokeFormatString or its bytes member is not readable.

VO_LIMIT_EXCEEDED

when the number of voInkStrokeBuilder instances would exceed the allowed limit (please remember that the limit can be shared by several types).
