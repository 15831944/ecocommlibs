insertArrayValueAt

bool insertArrayValueAt(voEngine engine, voJson array, int32_t index, voJson value)
Inserts an element to a JSON array.

If the given index is valid then the value is placed at the given position and the existing values starting from the index are shifted.

If the given index is -1 the value is inserted at the end of the array.

Parameters:
engine

the engine.

array

the target JSON array.

index

the index at which to place the new value. Use -1 to append at the end of the array.

value

the JSON value to insert.
Returns:

true on success, otherwise false.

Throws:
VO_NO_SUCH_ENGINE

when the engine reference is invalid.

VO_NO_SUCH_OBJECT

when the target reference is invalid.

VO_INVALID_OPERATION

when target is not a voJson of type VO_JSON_ARRAY.

VO_INDEX_OUT_OF_BOUNDS

when index is out of bounds or not -1.

VO_NO_SUCH_OBJECT

when the value reference is invalid.

VO_INVALID_OBJECT

when value is not a voJson.
