Value Detail

VO_USER_MATH_NODE_LIMIT

VO_USER_MATH_NODE_LIMIT
Represents the maximum number of voMathNode instances that can exist at the same time in the user side of a MyScript enable application.

The special value zero means no limit. Other values are adjusted to the next multiple of 16. A value under the current count is permitted and will constrain future resource creations.

Usage: 
uint32_t value;
voGetTypeProperty(engine, VO_MathNode, VO_USER_MATH_NODE_LIMIT, &value, sizeof(value));
voSetTypeProperty(engine, VO_MathNode, VO_USER_MATH_NODE_LIMIT, &value, sizeof(value));
