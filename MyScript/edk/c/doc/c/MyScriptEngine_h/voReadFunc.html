Type voReadFunc

voReadFunc
#include <c/MyScriptEngine.h>
typedef size_t (*voReadFunc)(void *buffer, size_t bufferLength, void *userParam)

Defines an application supplied function that performs read operations triggered by the voLoadObject() function.

Parameters:
buffer

a pointer to the destination/source buffer.

bufferLength

the length of the buffer in bytes.

userParam

an optional user defined parameter passed to the callback.
Returns:

the number of bytes actually read, or VO_INVALID_SIZE_T on failure.

Note:

When your implementation of the read function returns 0 on failure instead of the magic VO_INVALID_SIZE_T value, the voGetError() function will report the VO_PREMATURE_END_OF_DATA error instead of the VO_IO_FAILURE error.
