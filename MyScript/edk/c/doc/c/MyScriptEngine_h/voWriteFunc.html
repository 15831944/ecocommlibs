Type voWriteFunc

voWriteFunc
#include <c/MyScriptEngine.h>
typedef size_t (*voWriteFunc)(const void *buffer, size_t bufferLength, void *userParam)

Defines an application supplied function that performs write operations triggered by the voStoreObject() function.

Parameters:
buffer

a pointer to the destination/source buffer.

bufferLength

the length of the buffer in bytes.

userParam

an optional user defined parameter passed to the callback.
Returns:

the number of bytes actually written.
