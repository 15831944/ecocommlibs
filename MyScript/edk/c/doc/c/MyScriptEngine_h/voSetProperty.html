c/MyScriptEngine.h
Function voSetProperty

#include <c/MyScriptEngine.h>
bool voSetProperty(voEngine engine, voEngineObject object, voPropId property, const void *value, size_t sizeOfValue)

Sets or resets the value of the specified object property.

Parameters:
engine

the engine.

object

the object to be modified.

property

the identifier of the object property.

value

the recipient for the value. Use NULL when you want to reset the property to its default value.

sizeOfValue

the size of the recipient for the value, in bytes. This parameter is used by the engine to ensure that the recipient you are using has the proper type.
Returns:

true on success, otherwise false.

Throws:
VO_NO_SUCH_ENGINE

when the engine reference is invalid.

VO_NO_SUCH_OBJECT

when the object reference is invalid.

VO_NO_SUCH_PROPERTY

when the specified property does not exist for the specified object.

VO_READ_ONLY_PROPERTY

when the property is read-only.

VO_INVALID_SIZE

when the recipient does not have the proper size.

VO_INVALID_USER_BUFFER

when the given recipient is not readable.

VO_INVALID_ARGUMENT

when the given value is invalid.
