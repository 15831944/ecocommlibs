c/MyScriptEngine.h
Function voCloneObject

#include <c/MyScriptEngine.h>
voEngineObject voCloneObject(voEngine engine, voEngineObject object)

Clones the specified object.

Parameters:
engine

the engine.

object

the object to be cloned.
Returns:

a reference to a clone of the specified object on success, otherwise NULL.

Throws:
VO_NO_SUCH_ENGINE

when the engine reference is invalid.

VO_NO_SUCH_OBJECT

when the object reference is invalid.

VO_LIMIT_EXCEEDED

when the number of instances of the specified type would exceed the allowed limit (please remember that the limit can be shared by several types).

VO_INVALID_OPERATION

when object does not support cloning.
