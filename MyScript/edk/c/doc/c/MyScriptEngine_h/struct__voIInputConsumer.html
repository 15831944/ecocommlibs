Structure _voIInputConsumer
#include <c/MyScriptEngine.h>
struct _voIInputConsumer
 extends voIInput

Functions composing the voIInputConsumer interface.

An input consumer is a special input that consumes another input to produce any desired result. As it is itself an input, it can become the source of another consumer.
