struct _voString

Represents a character string encoded in a specific charset.

When a voString is used as an output by a function, the typical usage is to call the function twice:

  • once with bytes set to NULL in which case byteCount will be set to the number of bytes you have to allocate/provide.

  • a second time with bytes pointing a writable buffer and byteCount set to the maximum number of bytes the function is allowed to write.

  • in any case, on output byteCount will always be set to the number of bytes needed to hold the string even if you allocated less.
