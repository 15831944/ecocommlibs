Value Detail

VO_ENGINE_LIMIT

VO_ENGINE_LIMIT
Represents the maximum number of voEngine instances that can exist at the same time in a MyScript enabled application.

The special value zero means no limit. Other values are adjusted to the next multiple of 16. A value under the current count is permitted and will constrain future creations.

Usage: 
uint32_t value;
voGetTypeProperty(engine, VO_Engine, VO_ENGINE_LIMIT, &value, sizeof(value));
voSetTypeProperty(engine, VO_Engine, VO_ENGINE_LIMIT, &value, sizeof(value));

Note:

The engine parameter of vo[Get|Set]TypeProperty() must be set to NULL.

VO_DEFAULT_CHARSET

VO_DEFAULT_CHARSET
Holds a default voCharset object.

This property enables you to specify a default charset once and for all: when done, you are allowed to pass NULL to every function that takes a voCharset parameter.

Note:

Since version 5.0 voCreateEngine() initializes this with a UTF-8 charset, while previous versions did not set any default charset.

Usage: 
voCharset value;
voGetTypeProperty(engine, VO_Engine, VO_DEFAULT_CHARSET, &value, sizeof(value));
voSetTypeProperty(engine, VO_Engine, VO_DEFAULT_CHARSET, &value, sizeof(value));
