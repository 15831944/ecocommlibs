Value Detail

VO_USER_OBJ_LIMIT

VO_USER_OBJ_LIMIT
Represents the maximum number of voEngineObject instances that can exist at the same time in the user side of a MyScript enabled application.

The special value zero means no limit. Other values are adjusted to the next multiple of 16. A value under the current count is permitted and will constrain future creations.

Usage: 
uint32_t value;
voGetTypeProperty(engine, VO_EngineObject, VO_USER_OBJ_LIMIT, &value, sizeof(value));
voSetTypeProperty(engine, VO_EngineObject, VO_USER_OBJ_LIMIT, &value, sizeof(value));

Note:

This applies only to types that do not define their own user limit property.
