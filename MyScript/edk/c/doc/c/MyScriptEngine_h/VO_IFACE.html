c/MyScriptEngine.h
Define VO_IFACE

#include <c/MyScriptEngine.h>
#define VO_IFACE(engine, IName) ((const vo##IName*)voGetInterface(engine, VO_##IName))

Helper macro that facilitates the use of interface functions by performing the correct cast on the result of voGetInterface().

Parameters:
engine

the engine.

IName

the name of the interface without prefix, so that prepending VO_ gives the identifier and prepending vo gives the function structure type.
Example: 

VO_IFACE(engine, ICharset)->setDefaultCharacter(engine, charset, '?');

Note:

Invoking an interface function with an instance of a type that does not implement this interface sets the error code to VO_INVALID_OPERATION.
