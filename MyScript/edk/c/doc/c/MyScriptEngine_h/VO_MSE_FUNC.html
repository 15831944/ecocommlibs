#include <c/MyScriptEngine.h>
VO_MSE_FUNC(voEngineObject, voAcquireObject,(voEngine engine, voEngineObject object))

Returns a new reference to the specified object.

This function is useful in some languages like C++ where you need to be able to design copyable objects.

Example: 

class EngineObject
{
public:
  EngineObject(voEngine engine, voTypeId type) : e(engine), o(voCreateObject(engine, type)) {}
  EngineObject(const EngineObject& rhs) : e(rhs.e), o(voAcquireObject(rhs.e, rhs.o)) {}

...

  ~EngineObject() { voReleaseObject(e, o); }
private:
  voEngine e;
  voEngineObject o;
};

Parameters:
engine

the engine.

object

the object to be acquired.
Returns:

a new reference to the specified object on success, otherwise NULL.

Throws:
VO_NO_SUCH_ENGINE

when the engine reference is invalid.

VO_NO_SUCH_OBJECT

when the object reference is invalid.

VO_LIMIT_EXCEEDED

when the number of user references to object would exceed 65535.
See also:
