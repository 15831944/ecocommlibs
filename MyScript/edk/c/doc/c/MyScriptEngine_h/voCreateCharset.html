c/MyScriptEngine.h
Function voCreateCharset

#include <c/MyScriptEngine.h>
VO_INLINE voCharset voCreateCharset(voEngine engine, const char *name)

Creates a new voCharset instance.

Parameters:
engine

the engine.

name

the name of the charset to be created. Accepted values are: "US-ASCII", "ISO-8859-1", "ISO-8859-15", "windows-1252", "UTF-8", "UTF-8Y", "UTF-16", "UTF-16BE", "UTF-16LE", "UTF-32", "UTF-32BE", "UTF-32LE". If NULL, "UTF-8" is selected.
Returns:

the newly created voCharset instance on success, otherwise NULL.

Throws:
VO_NO_SUCH_ENGINE

when the engine reference is invalid.

VO_INVALID_ARGUMENT

when name is invalid.

VO_LIMIT_EXCEEDED

when the number of voCharset instances would exceed the allowed limit (please remember that the limit can be shared by several types).
