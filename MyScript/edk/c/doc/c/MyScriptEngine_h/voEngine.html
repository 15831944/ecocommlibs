typedef voReference voEngine

voEngine reference type.

Represents the MyScript runtime execution environment.

Note:

The voEngine type differs from the voEngineObject type and as a consequence will cause a VO_NO_SUCH_OBJECT error if you try to use a voEngine instead of a voEngineObject.
