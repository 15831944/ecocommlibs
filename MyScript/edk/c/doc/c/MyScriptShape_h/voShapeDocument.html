commitResult

bool voIShapeDocument::commitResult(voEngine engine, voShapeDocument target, const voShapeInkRange *ranges, int32_t rangeCount)
Marks a portion of the shape document segments as committed.

Note:

The shape recognition results associated with committed portions of a document are used as input of the next shape recognition process, rather than the corresponding input strokes.

Parameters:
engine

the engine.

target

the target document.

ranges

an array of ink ranges, or NULL to commit the entire result.

rangeCount

the number of ink ranges values in ranges.
Returns:

true on success, otherwise false.

Throws:
VO_NO_SUCH_ENGINE

when the engine reference is invalid.

VO_NO_SUCH_OBJECT

when the target reference is invalid.

VO_INVALID_OPERATION

when target is not a voShapeDocument.

VO_INVALID_USER_BUFFER

when ranges is not readable.

VO_INVALID_ARGUMENT

when ranges is not NULL and rangeCount <= 0

VO_INVALID_ARGUMENT

when ranges does not match the content of the document.

7.0_0

getStrokeAt

bool voIShapeDocument::getStrokeAt(voEngine engine, voShapeDocument target, int32_t index, float *x, size_t xByteStride, float *y, size_t yByteStride, int32_t *pointCount)
Returns the points of the stroke at the specified index.

Parameters:
engine

the engine.

target

the target document.

index

the index of the requested stroke.

x

the recipient for the x values.

xByteStride

the stride in bytes between each x value.

y

the recipient for the y values.

yByteStride

the stride in bytes between each y value.

pointCount

on input, the capacity of the x and y recipients, on output, the recipient for the number of points in the stroke.
Returns:

true on success, otherwise false.

Throws:
VO_NO_SUCH_ENGINE

when the engine reference is invalid.

VO_NO_SUCH_OBJECT

when the target reference is invalid.

VO_INVALID_OPERATION

when target is not a voShapeDocument.

VO_INDEX_OUT_OF_BOUNDS

when index is out of bounds.

VO_INVALID_USER_BUFFER

when pointCount is not readable or writable.

VO_INVALID_ARGUMENT

when x or y is not NULL and pointCount <= 0

VO_INVALID_USER_BUFFER

when the combination of x, xByteStride, y, yByteStride and pointCount leads to invalid memory access.
Note:

You can pass NULL in place of x and y to retrieve the number of points in the stroke.

setCoordinateResolution

bool voIShapeDocument::setCoordinateResolution(voEngine engine, voShapeDocument target, float resolution)
Sets the resolution of the coordinates in dpi.

This is the resolution of the grid of integer values in the chosen coordinate system. For example, if you use floating-point pixel coordinates you must give the screen resolution, no matter the resolution of the capturing device.

Note:

The value 0 resets to the default behavior of automatic estimation.

Parameters:
engine

the engine.

target

the target document.

resolution

the coordinate resolution.
Returns:

true on success, otherwise false.
