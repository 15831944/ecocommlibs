Returns the model at the specified index in the shape knowledge attached to this recognizer.

Parameters:

the engine.

target

the target recognizer.

index

the index of the model to be retrieved.
Returns:

the model on success, otherwise NULL.

Throws:
VO_NO_SUCH_ENGINE

when the engine reference is invalid.

when index is out of bounds.
