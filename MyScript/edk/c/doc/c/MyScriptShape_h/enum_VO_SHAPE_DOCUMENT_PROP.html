VO_SHAPE_DOCUMENT_DISABLE_AUTO_COMMIT_RESULT

VO_SHAPE_DOCUMENT_DISABLE_AUTO_COMMIT_RESULT
Tells the shape document to not automatically commit its result, i.e.

once a document is processed by a shape recognizer, the recognition result is not used as input for the next recognition.
