VO_SHAPE_BEAUTIFIER_ALIGNMENT_STRENGTH

VO_SHAPE_BEAUTIFIER_ALIGNMENT_STRENGTH
A value between 0.0f and 1.0f that adjusts the beautification alignment strength of the beautifier.

@xmlonly <property type="ShapeBeautifier" read-only="no">\endxmlonlyfloat\xmlonly</property>\endxmlonly

Note:

A value equal to 0.0f disables the alignment constraints of the beautification process. A value close to 1.0f increases the likelihood of shapes being aligned with the vertical and horizontal document axis.

VO_SHAPE_BEAUTIFIER_CONGRUENCE_STRENGTH

VO_SHAPE_BEAUTIFIER_CONGRUENCE_STRENGTH
A value between 0.0f and 1.0f that adjusts the beautification congruence strength of the beautifier.

@xmlonly <property type="ShapeBeautifier" read-only="no">\endxmlonlyfloat\xmlonly</property>\endxmlonly

Note:

A value equal to 0.0f disables the congruence constraints of the beautification process. A value close to 1.0f increases the likelihood of shapes being of the same size and aligned with each other.

VO_SHAPE_BEAUTIFIER_CONNECTION_STRENGTH

VO_SHAPE_BEAUTIFIER_CONNECTION_STRENGTH
