c/MyScriptShape.h
Function voCreateShapeDecoratedEllipticArc

#include <c/MyScriptShape.h>
VO_INLINE voShapeDecoratedEllipticArc voCreateShapeDecoratedEllipticArc(voEngine engine, const voShapeDecoratedEllipticArcData *data)

Creates a new voShapeDecoratedEllipticArc instance.

Parameters:
engine

the engine.

data

the data of the decorated elliptic arc to be created.
Returns:

the newly created voShapeDecoratedEllipticArc instance on success, otherwise NULL.

Throws:
VO_NO_SUCH_ENGINE

when the engine reference is invalid.

VO_INVALID_USER_BUFFER

when data is not readable.

VO_LIMIT_EXCEEDED

when the number of voShapeDecoratedEllipticArc instances would exceed the allowed limit (please remember that the limit can be shared by several types).
