VO_USER_MUSIC_ELEMENT_LIMIT

VO_USER_MUSIC_ELEMENT_LIMIT
Represents the maximum number of voMusicElement instances that can exist at the same time in the user side of a MyScript enabled application.

The special value zero means no limit. Other values are adjusted to the next multiple of 16. A value under the current count is permitted and will constrain future resource creations.

Usage: 
uint32_t value;
voGetTypeProperty(engine, VO_MusicElement, VO_USER_MUSIC_ELEMENT_LIMIT, &value, sizeof(value));
voSetTypeProperty(engine, VO_MusicElement, VO_USER_MUSIC_ELEMENT_LIMIT, &value, sizeof(value));
